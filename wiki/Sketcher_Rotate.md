# Sketcher Rotate

## Description

The <img alt="" src=images/Sketcher_Rotate.svg  style="width:24px;"> [Sketcher Rotate](Sketcher_Rotate.md) tool rotates and optionally copies selected edges. Copies are evenly distributed according to the rotation angle. When starting the tool, the mouse pointer changes to a white cross with a red rotate icon:  <img alt="" src=images/Sketcher_Pointer_Create_Rotate.svg  style="width:32px;"> .
