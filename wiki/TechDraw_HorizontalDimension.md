# TechDraw HorizontalDimension

## Description

The **TechDraw HorizontalDimension** tool adds a horizontal dimension to a View. The dimension may be the horizontal distance between two points or the horizontal length of a straight edge. As with the [TechDraw LengthDimension](TechDraw_LengthDimension.md) tool it is also possible to select two edges or a point and an edge, but these options make less sense for this tool.

For more information see [TechDraw LengthDimension](TechDraw_LengthDimension.md).
