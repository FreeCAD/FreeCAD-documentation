# SheetMetal UnfoldUpdate

## Description

The <img alt="" src=images/SheetMetal_UnfoldUpdate.svg  style="width:24px;"> [Unfold Update](SheetMetal_UnfoldUpdate.md) command updates all unfold objects.

## Usage

1.  There are several ways to invoke the command:
    -   Press the **<img src="images/SheetMetal_UnfoldUpdate.svg_" width=16px> [Unfold Update](SheetMetal_UnfoldUpdate.md)** button.
    -   Select the **Sheet Metal → <img src="images/SheetMetal_UnfoldUpdate.svg" width=16px> Unfold Update** option from the menu.
    -   Right-click in the [Tree view](Tree_view.md) or the [3D view](3D_view.md) and select the **Sheet Metal → <img src="images/SheetMetal_UnfoldUpdate.svg" width=16px> Unfold Update** option from the context menu.
2.  All **Unfold** objects are updated.
