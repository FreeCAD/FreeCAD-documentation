# TechDraw HoleShaftFit

## Description

The **Hole/Shaft Fit** tool adds hole/shaft tolerances using ISO 286 to a length dimension or a diameter dimension.

 <img alt="" src=images/TechDraw_HoleShaftFitExample.png  style="width:500px;">  
*Left the opened panel, right some added tolerances *

## Usage

1.  Select one length dimension (horizontal, vertical or oblique) or one circle diameter dimension.
2.  Press the **<img src="images/TechDraw_HoleShaftFit.svg" width=16px> [Hole/Shaft fit](TechDraw_HoleShaftFit.md)** button.
3.  A panel opens in the Combo View.
4.  Select shaft fit / hole fit to show whether your dimension refers to a shaft or a hole.
5.  Select the desired ISO 286 fit field using the combo box. The partner field and the fit type (loose, snug, press) is shown.
6.  Press the OK button.

The label of the selected field and the calculated tolerance values are added to the dimension and shown in the drawing.
