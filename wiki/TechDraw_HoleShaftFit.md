# TechDraw HoleShaftFit

## Description

The **TechDraw HoleShaftFit** tool adds hole or shaft tolerances using ISO 286 to a linear dimension or a diameter dimension.

 <img alt="" src=images/TechDraw_HoleShaftFitExample.png  style="width:500px;">  
*On the left the task panel, on the right some added tolerances*

## Usage

1.  Select a single linear dimension ([length](TechDraw_LengthDimension.md), [horizontal](TechDraw_HorizontalDimension.md) or [vertical](TechDraw_VerticalDimension.md)) or a single [diameter](TechDraw_DiameterDimension.md) dimension.
2.  There are several ways to invoke the tool:
    -   Press the **<img src="images/TechDraw_HoleShaftFit.svg" width=16px> [Add hole or shaft fit](TechDraw_HoleShaftFit.md)** button.
    -   Select the **TechDraw → Symbols → <img src="images/TechDraw_HoleShaftFit.svg" width=16px> Add hole or shaft fit** option from the menu.
3.  A task panel opens.
4.  Specify **shaft fit** or **hole fit** as applicable.
5.  Select the desired ISO 286 fit option from the combo box. The corresponding field and the fit type (loose fit, snug fit or press fit) are updated.
6.  Press the **OK** button.
7.  The selected option and the calculated tolerance values are added to the dimension.
