# BIM TDView

## Description

The <img alt="" src=images/BIM_TDView.svg  style="width:24px;"> [BIM TDView](BIM_TDView.md) tool creates a new TechDraw view from a section plane or 2D objects.

1.  There are several ways to invoke the tool:
