# ImageConv/pt-br
O **ImageConv** é um programa usado para converter imagens de vários formatos para o formato XPM, que pode ser compilado no FreeCAD.



## Construção

O ImageConv está localizado na árvore de código-fonte do FreeCAD:


{{Incode|/trunk/src/Tools/ImageTools/ImageConv}}

A criação é fácil com o \"Makefile\" ou com o VisualStudio-Project.



## Utilização

O ImageConv é uma ferramenta de linha de comando.
