# CAM Comment/pt-br

## Description

The tool <img alt="" src=images/CAM_Comment.svg  style="width:24px;"> [Comment](CAM_Comment.md) inserts a comment. When exporting a project to G-code, the comments will be inserted in the G-code program, and can be read by people reading the file. Some machine controllers will also display comments on their display screens.

## Usage

1.  Press the **<img src="images/CAM_Comment.svg" width=16px> [Comment](CAM_Comment.md)** button or use the **P** then **C** keyboard shortcut.
2.  Write your comment in the **Comment** property

## Properties

-    **Comment**: The text of your comment
