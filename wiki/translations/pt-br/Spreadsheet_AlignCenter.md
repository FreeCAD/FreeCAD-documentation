# Spreadsheet AlignCenter/pt-br

## Description

The <img alt="" src=images/Spreadsheet_AlignCenter.svg  style="width:24px;"> **Spreadsheet AlignCenter** tool aligns the contents of selected cells to the center horizontally.

## Usage

1.  Make sure there is an active [spreadsheet](Spreadsheet_CreateSheet.md).
2.  Select one or more cells.
3.  There are several ways to invoke the tool:
    -   Press the **[<img src=images/Spreadsheet_AlignCenter.svg style="width:16px"> [Align center](Spreadsheet_AlignCenter.md)** button.
    -   Select the **Spreadsheet → Alignment → [<img src=images/Spreadsheet_AlignCenter.svg style="width:16px"> Align center** option from the menu.
