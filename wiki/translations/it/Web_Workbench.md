# <img alt="L\'icona di Web" src=images/Workbench_Web.svg  style="width:64px;"> Web Workbench/it

## Introduzione

L\' [Ambiente Web](Web_Workbench/it.md) non è un vero ambiente di lavoro, ma fornisce una finestra del browser all\'interno di FreeCAD.

Da questo browser è possibile navigare sul Web, fare clic sui collegamenti, leggere la documentazione online e accedere ai siti Web (sono supportati i cookie). L\'Ambiente Web è anche una base per lo sviluppo di strumenti relativi al Web in FreeCAD. La finestra del browser è basata su [QtWebEngine](https://wiki.qt.io/QtWebEngine) e fornisce pulsanti di navigazione standard come stop, ricarica, pagina precedente e successiva e zoom avanti e indietro.

## Strumenti

L\'Ambiente Web offre i seguenti strumenti:

-   <img alt="" src=images/Web_BrowserSetURL.svg  style="width:32px;"> [Imposta URL](Web_BrowserSetURL/it.md): cambia l\'URL della scheda Web attiva.

-   <img alt="" src=images/Web_OpenWebsite.svg  style="width:32px;"> [Apri sito Web\...](Web_OpenWebsite/it.md): Apre il [sito Web FreeCAD](https://freecadweb.org).

-   <img alt="" src=images/Web_BrowserBack.svg  style="width:32px;"> [Pagina precedente](Web_BrowserBack/it.md): va alla pagina web precedente.

-   <img alt="" src=images/Web_BrowserNext.svg  style="width:32px;"> [Pagina successiva](Web_BrowserNext/it.md): va alla pagina Web successiva.

-   <img alt="" src=images/Web_BrowserRefresh.svg  style="width:32px;"> [Aggiorna pagina web](Web_BrowserRefresh/it.md): aggiorna la pagina web corrente.

-   <img alt="" src=images/Web_BrowserStop.svg  style="width:32px;"> [Interrompi caricamento](Web_BrowserStop/it.md): interrompe il caricamento della pagina Web corrente.

-   <img alt="" src=images/Web_BrowserZoomIn.svg  style="width:32px;"> [Zoom in](Web_BrowserZoomIn/it.md): ingrandisce la pagina Web corrente.

-   <img alt="" src=images/Web_BrowserZoomOut.svg  style="width:32px;"> [Zoom indietro](Web_BrowserZoomOut/it.md): Rimpicciolisce la pagina Web corrente.
