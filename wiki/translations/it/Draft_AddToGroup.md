# Draft AddToGroup/it


</div>

## Description


<div class="mw-translate-fuzzy">

## Descrizione

Questo strumento visualizza un menu a discesa che consente di aggiungere gli oggetti selezionati a un gruppo esistente, o di rimuoverli dal loro gruppo corrente.


</div>

In FreeCAD version 0.20 the command can also handle group-like [Arch](Arch_Workbench.md) objects.

## Usage


<div class="mw-translate-fuzzy">

## Utilizzo

1.  Selezionare gli oggetti che si desidera organizzare.
2.  Fare clic con il tasto destro e selezionare **Utilità → <img src="images/Draft_AddToGroup.svg" width=16px> [Aggiungi al gruppo...](Draft_AddToGroup/it.md)**.
3.  Selezionare il gruppo desiderato o fare clic su **Ungroup**.
