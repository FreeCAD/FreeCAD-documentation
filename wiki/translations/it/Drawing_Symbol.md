### Descrizione

Questo comando consente di aggiungere il contenuto di un\'immagine SVG in una [pagina di disegno](Drawing_Landscape_A3/it.md) selezionata. Questo contenuto può poi essere spostato e riscalato sulla pagina. Il contenuto dell\'immagine SVG viene copiato nel documento FreeCAD, quindi è indipendente dal file originale, e viene visualizzato nello stesso modo anche su un altro computer che non dispone del file SVG originale.

## Utilizzo

1.  Creare una [pagina di disegno](Drawing_Landscape_A3/it.md)
2.  [Aggiornare](Std_Refresh/it.md) la vista se il disegno non viene aperto
3.  Premere il pulsante **<img src="images/Drawing_Symbol.png" width=16px> [Simbolo](Drawing_Symbol/it.md)
**
4.  Selezionare un file SVG
5.  Regolare le proprietà necessarie, ad esempio, la posizione e la scala.
