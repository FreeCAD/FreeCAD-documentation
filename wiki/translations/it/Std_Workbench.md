# Std Workbench/it



## Descrizione

Il comando **Ambienti** attiva l\'[ambiente di lavoro](Workbenches/it.md) selezionato.

![](images/Std_Workbench_ComboBox_Icons_And_Text.png ) 
*Il selettore ComboBox predefinito dell'ambiente di lavoro*

![](images/Std_Workbench_TabBar_Icons_Only.png ) 
*Il selettore opzionale TabBar degli ambienti di lavoro (qui visualizzato solo con icone) {{Version/it|1.0*}}



## Utilizzo

1.  Esistono diversi modi per richiamare il comando:
    -   Selezionare un ambiente di lavoro dal ComboBox o dalla TabBar ({{Version/it|1.0}}) nella barra degli strumenti dell\'ambiente di lavoro.
    -   Premere il pulsante **<img src="images/List-add.svg" width=x16px><img src="images/Toolbar_flyout_arrow.svg" width=x16px>** sulla TabBar per selezionare un ambiente di lavoro che è stato disabilitato nella [preferenza](Preferences_Editor/it#Available_Workbenches.md) dal menu che si apre.
    -   Selezionare un ambiente di lavoro dal sottomenu **Visualizza → Ambiente di lavoro**.



## Note

-   Ulteriori [ambienti complementari](External_Workbenches/it.md) possono essere scaricati con <img alt="" src=images/Std_AddonMgr.svg  style="width:24px;"> [Addon Manager](Std_AddonMgr/it.md).



## Preferenze

Vedere anche: [Editor delle Preferenze](Preferences_Editor/it.md).

-   È disponibile una [pagina con le preferenze correlate](Preferences_Editor/it#Available_Workbenches.md): **Modifica → Preferenze... → Ambienti di lavoro → Ambienti di lavoro disponibili**. E\' possibile modificare l\'**Ambiente di avvio**, il **Selettore dell'ambiente di lavoro** e altro ancora. Il pulsante **<img src="images/List-add.svg" width=x16px><img src="images/Toolbar_flyout_arrow.svg" width=x16px>** sulla TabBar dà accesso a questa pagina tramite il suo menu.



## Script

Vedere anche: [Autogenerated API documentation](https://freecad.github.io/SourceDoc/) e [Script di base per FreeCAD](FreeCAD_Scripting_Basics/it.md).

Per cambiare l\'ambiente usare il metodo `activWorkbench` del modulo FreeCADGui.


```python
import FreeCADGui

FreeCADGui.activateWorkbench("PartDesignWorkbench")
```
