## Description


<div class="mw-translate-fuzzy">

#### Descrizione

Esporta un file di programmazione del robot.
Si può esportare una subroutine KUKA compatta o una subroutine KUKA completa.


</div>


<div class="mw-translate-fuzzy">

#### Utilizzo

1.  Selezionare il robot e la traiettoria nella vista ad albero.
2.  Scegliere ** Robot** → ** Esporta traiettoria** → **<img src="images/Robot_Export.png" width=32px> Subroutine Kuka compatta...** dal menu in alto.

oppure

1.  Selezionare il robot e la traiettoria nella vista ad albero.
2.  Scegliere ** Robot** → ** Esporta traiettoria** → **<img src="images/Robot_Export.png" width=32px> Subroutine Kuka completa...** dal menu in alto.


</div>

1.  Select robot and trajectory in the [Tree view](Tree_view.md).
2.  There are several ways to Export trajectory:
    -   Choose **Robot** → **Export trajectory** → **<img src="images/Robot_Export.svg" width=32px> Kuka compact subroutine...** from the top menu.
    -   Choose **Robot** → **Export trajectory** → **<img src="images/Robot_Export.svg" width=32px> Kuka full subroutine...** from the top menu.


<div class="mw-translate-fuzzy">

Viene creato un file KRL (\*.src).


</div>
