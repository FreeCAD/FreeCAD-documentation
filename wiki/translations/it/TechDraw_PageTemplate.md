# TechDraw PageTemplate/it



## Descrizione

Lo strumento **TechDraw Pagina da modello** crea un nuovo oggetto Pagina utilizzando il file di un modello selezionato in una finestra di dialogo.

La directory di partenza per il dialogo può essere specificata nelle [Preferenze di TechDraw](TechDraw_Preferences/it.md)

<img alt="" src=images/A4_Landscape_ISO7200_Pep.svg  style="width:400px;"> 
*Uno dei modelli che viene fornito con TechDraw: A4_Landscape_ISO7200_Pep.svg*



## Utilizzo

1.  Deve esistere un documento attivo.
2.  Esistono diversi modi per richiamare lo strumento:
    -   Premere il pulsante **<img src="images/TechDraw_PageTemplate.svg" width=16px> [Inserisci Pagina utilizzando un modello](TechDraw_PageTemplate/it.md)**.
    -   Selezionare l\'opzione **TechDraw → Pagina → <img src="images/TechDraw_PageTemplate.svg" width=16px> Inserisci Pagina utilizzando un modello** dal menu.



## Proprietà

Vedere [TechDraw Pagina predefinita](TechDraw_PageDefault/it#Proprietà.md).



## Script

Vedere anche: [Autogenerated API documentation](https://freecad.github.io/SourceDoc/) e [Script di base per FreeCAD](FreeCAD_Scripting_Basics/it.md).

Una Pagina basata su un modello selezionato può essere creata con [macro](Macros/it.md) e dalla console [Python](Python/it.md) utilizzando le seguenti funzioni:


```python
import FreeCAD as App
from PySide import QtGui

doc = App.ActiveDocument
default_dir = App.getResourceDir() + "Mod/TechDraw/Templates"
param = App.ParamGet("User parameter:BaseApp/Preferences/Mod/TechDraw/Files")
template_dir = param.GetString("TemplateDir", default_dir)

template_file = QtGui.QFileDialog.getOpenFileName(QtGui.QApplication.activeWindow(),
                                                  "Select a Template File", 
                                                  template_dir,
                                                  "Template (*.svg)")
                                                  
page = doc.addObject("TechDraw::DrawPage", "Page")
template = doc.addObject("TechDraw::DrawSVGTemplate", "Template")
template.Template = template_file[0]
page.Template = template

doc.recompute()
```



### Campi di testo modificabili 


**Vedere anche:**

[Modelli di squadrature](TechDraw_Templates/it.md) per ulteriori informazioni sulla creazione di modelli .

Una volta creata una nuova pagina, il suo attributo `Template` contiene un dizionario `EditableTexts` con il nome dei campi modificabili (chiavi) e i loro valori testuali. Copiare questo dizionario in una variabile, apportare le modifiche, quindi riassegnare il dizionario all\'attributo `EditableTexts` per vedere le modifiche.


```python
page = FreeCAD.ActiveDocument.Page
texts = page.Template.EditableTexts

for key, value in texts.items():
    print("{0} = {1}".format(key, value))

texts["FC-Title"] = "The title of my page"
page.Template.EditableTexts = texts
```
