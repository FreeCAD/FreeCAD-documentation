# TechDraw ComplexSection/it



## Descrizione

Lo strumento **TechDraw Sezione complessa** inserisce una vista in sezione trasversale basata su una vista della parte esistente e un profilo.

<img alt="" src=images/TechDraw_QuarterSection_example.png  style="width:350px;"> 
*Una vista in sezione di un quarto creata con lo strumento Sezione complessa*

<img alt="" src=images/TechDraw_AlignedSection_example.png  style="width:350px;"> 
*Una vista in sezione Deviata creata con lo strumento Sezione complessa*

<img alt="" src=images/TechDraw_OffsetSection_example.png  style="width:350px;"> 
*Una vista in sezione Sfalsata creata con lo strumento Sezione complessa*



## Utilizzo

1.  Selezionare una vista della parte e un oggetto profilo nella [Vista 3D](3D_view/it.md) o nella [Vista ad albero](Tree_view/it.md). I profili sono in genere schizzi, ma funzionerà qualsiasi oggetto la cui forma possa essere trasformata in una polilinea.
2.  Esistono diversi modi per richiamare lo strumento:
    -   Premere il pulsante **<img src="images/TechDraw_ComplexSection.svg" width=16px> [Inserisci Sezione complessa](TechDraw_ComplexSection/it.md)**.
    -   Selezionare l\'opzione **TechDraw → Viste TechDraw → <img src="images/TechDraw_ComplexSection.svg" width=16px> Inserisci Sezione complessa** dal menu.
3.  Si apre un pannello delle azioni che aiuterà a calcolare le varie proprietà. Vengono calcolati valori ragionevoli per la direzione della vista, ma questi possono essere modificati.

<img alt="" src=images/TechDraw_ComplexSection_Taskview1.png  style="width:" height="380px;"> <img alt="" src=images/TechDraw_ComplexSection_Taskview2.png  style="width:" height="380px;">



## Proprietà

Una Sezione Complessa, formalmente un oggetto {{Incode|TechDraw::DrawComplexSection}}, deriva da un oggetto [Vista Sezione](TechDraw_SectionView/it#Properietà.md), formalmente un oggetto {{Incode|TechDraw::DrawViewSection}} ed eredita tutti le sue proprietà. Ha inoltre le seguenti proprietà aggiuntive:



### Dati


{{TitleProperty|Cutting Tool}}

-    **Cutting Tool Wire Object**: l\'oggetto documento la cui forma verrà utilizzata per generare il profilo di taglio.

-    **Projection Strategy**: controlla come viene eseguito il taglio e come viene proiettato il risultato sulla pagina:

    -   
        {{Value|Offset}}
        
        : esegue un taglio semplice della forma sorgente e proietta il risultato.

    -   
        {{Value|Aligned}}
        
        : Taglia la forma di origine utilizzando uno strumento creato da ciascun segmento (bordo) del profilo di taglio. I risultati di ogni taglio vengono proiettati in una serie verticale o orizzontale, a seconda dell\'orientamento del profilo di taglio.

    -   
        {{Value|NoParallel}}
        
        : come allineato, ma i segmenti del profilo paralleli alla direzione della vista vengono saltati.

## Script

Vedere anche: [Autogenerated API documentation](https://freecad.github.io/SourceDoc/) e [Script di base per FreeCAD](FreeCAD_Scripting_Basics/it.md).

Lo strumento Sezione Complessa può essere utilizzato nelle [macro](Macros/it.md) e dalla console [Python](Python/it.md) tramite le seguenti funzioni:


```python
doc = FreeCAD.ActiveDocument
box = doc.Box
profile = doc.Sketch
page = doc.Page

view = doc.addObject("TechDraw::DrawViewPart", "View")
page.addView(view)
view.Source = box
view.Direction = (0, 0, 1)

section = doc.addObject("TechDraw::DrawComplexSection", "ComplexSection")
page.addView(section)
section.BaseView = view
section.CuttingToolWireObject = profile
section.Direction = (0, 1, 0)
section.SectionNormal = (-1, 0, 0)
```
