# Start Workbench/it
**L''''Ambiente Start''' non è più incluso dalla versione 0.21.**

<img alt="L\'icona di Start" src=images/Workbench_Start.svg  style="width:128px;">

L\'[Ambiente Start](Start_Workbench.md) <img alt="" src=images/Workbench_Start.svg  style="width:24px;"> non è propriamente un ambiente, è solo la pagina che viene presentata quando si apre FreeCAD senza nessun documento caricato.

Da questa interfaccia è possibile creare nuovi file, passare agli ambienti di lavoro comunemente utilizzati con i pulsanti di avvio rapido ({{Version/it|1.0}}), visualizzare informazioni utili, passare ai file aperti di recente, visualizzare esempi, visualizzare la documentazione e ottenere il ultime novità dal mondo FreeCAD. È possibile modificare il comportamento di Start o cambiare l\'ambiente predefinito da caricare all\'avvio nel [Editor delle preferenze](Preferences_Editor/it.md) nel menu **Modifica → Preferenze → Start**.
