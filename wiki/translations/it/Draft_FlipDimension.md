# Draft FlipDimension/it



## Descrizione

Il comando <img alt="" src=images/Draft_FlipDimension.svg  style="width:24px;"> **Draft Capovolgi quota** ruota il testo delle [Draft Quote](Draft_Dimension/it.md) selezionate di 180° intorno alla linea di quota. Può essere utilizzato per correggere quote il cui testo appare speculare. Il comando non funziona correttamente per le quote angolari.



## Utilizzo

1.  Selezionare una o più [Draft Quote](Draft_Dimension.md).
2.  Esistono diversi modi per invocare il comando:
    -   Premere il pulsante **<img src="images/Draft_FlipDimension.svg" width=16px> [Capovolgi quota](Draft_FlipDimension/it.md)**.
    -   Selezionare l\'opzione **Modifiche → <img src="images/Draft_FlipDimension.svg" width=16px> Capovolgi quota** dal menu.



## Note

-   Le [Draft Quote](Draft_Dimension/it.md) hanno anche una proprietà **Flip Text**. Quando impostato su `True` il testo viene ruotato di 180° rispetto alla direzione normale. Questo può essere combinato con l\'effetto di questo comando.



## Script

Vedere anche: [Autogenerated API documentation](https://freecad.github.io/SourceDoc/) e [Script di base per FreeCAD](FreeCAD_Scripting_Basics/it.md).

Per capovolgere una [Draft Quota](Draft_Dimension/it.md) invertire la sua proprietà `Normal`.

Esempio:


```python
import FreeCAD as App
import Draft

doc = App.newDocument()

p1 = App.Vector(0, 0, 0)
p2 = App.Vector(1000, 0, 0)
p3 = App.Vector(500, 300, 0)
dimension = Draft.make_dimension(p1, p2, p3)
dimension.ViewObject.FontSize = 200

dimension.Normal = dimension.Normal.negative()
doc.recompute()
```
