# Draft WireToBSpline/it



## Descrizione

Il comando <img alt="" src=images/Draft_WireToBSpline.svg  style="width:24px;"> **Draft Polilinea in B-spline** converte [Draft Polilinee](Draft_Wire/it.md) in [Draft B-splines](Draft_BSpline/it.md) e viceversa.

<img alt="" src=images/Draft_Wire2BSpline_example.jpg  style="width:400px;"> 
*Conversione di un Draft Polilinea in un Draft B-spline e di una Draft B-spline chiusa in una Draft Polilinea chiusa*



## Utilizzo

1.  Selezionare una [Draft Polilinea](Draft_Wire/it.md) o una [Draft B-spline](Draft_BSpline/it.md).
2.  Ci sono diversi modi per invocare il comando:
    -   Premere il pulsante **<img src="images/Draft_WireToBSpline.svg" width=16px> [Draft Polilinea in B-spline](Draft_WireToBSpline.md)**.
    -   Selezionare l\'opzione **Modifiche → <img src="images/Draft_WireToBSpline.svg" width=16px> Polilinea in B-spline** dal menu.
3.  Viene creato un nuovo oggetto.



## Note

-   Il comando potrebbe far risultare una [Draft Polilinea](Draft_Wire/it.md) o [Draft B-spline](Draft_BSpline/it.md) autointersecante con una faccia. Tale oggetto non verrà visualizzato correttamente nella [Vista 3D](3D_view/it.md). La sua proprietà **Make Face**, o la sua proprietà **Closed**, deve essere impostata su `False`.



## Script

Vedere anche: [Autogenerated API documentation](https://freecad.github.io/SourceDoc/) e [Script di base per FreeCAD](FreeCAD_Scripting_Basics/it.md).

Per convertire una Polilinea in una B-spline, o viceversa, passare la proprietà `Points` dell\'oggetto sorgente al metodo `[make_bspline](Draft_BSpline/it#Script.md)`, o rispettivamente al metodo {{ incode\|[make_wire](Draft_Wire/it#Script.md)}}, del modulo Draft.

Esempio:


```python
import FreeCAD as App
import Draft

doc = App.newDocument()

p1 = App.Vector(1000, 1000, 0)
p2 = App.Vector(2000, 1000, 0)
p3 = App.Vector(2500, -1000, 0)
p4 = App.Vector(3500, -500, 0)

base_wire = Draft.make_wire([p1, p2, p3, p4])
base_spline = Draft.make_bspline([-p1, -1.3*p2, -1.2*p3, -2.1*p4])

points1 = base_wire.Points
spline_from_wire = Draft.make_bspline(points1)

points2 = base_spline.Points
wire_from_spline = Draft.make_wire(points2)

doc.recompute()
```
