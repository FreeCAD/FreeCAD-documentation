# Assembly CreateJointRevolute/de



## Beschreibung

Das Werkzeug <img alt="" src=images/Assembly_CreateJointRevolute.svg  style="width:24px;"> [Assembly DrehverbindungErstellen](Assembly_CreateJointRevolute/de.md) erstellt eine Drehverbindung (auch Scharnier genannt) zwischen zwei ausgewählten Bauteilen, die eine Drehung um eine gemeinsame Achse ermöglicht.



## Anwendung

1.  Wahlweise zwei geometrische Elemente zweier unterschiedlicher Formen auswählen. Andere Auswahlen werden verworfen.
2.  Es gibt mehrere Möglichkeiten, das Werkzeug aufzurufen:
    -   Die Schaltfläche **<img src="images/Assembly_CreateJointRevolute.svg" width=16px> [Drehverbindung erstellen](Assembly_CreateJointRevolute/de.md)** drücken.
    -   Den Menüeintrag **Assembly → <img src="images/Assembly_CreateJointRevolute.svg" width=16px> Drehverbindung erstellen** auswählen.
    -   Das Tastaturkürzel **R**.
3.  Der Dialog **Verbindung erstellen** wird im [Aufgaben-Fenster](Task_panel/de.md) geöffnet und listet die vorausgewählten Elemente.
4.  Für weitere Schritte siehe [Assembly StarreVerbindungErstellen](Assembly_CreateJointFixed/de#Anwendung.md).



## Eigenschaften

Siehe auch: [Eigenschafteneditor](Property_editor/de.md).

Ein **Revolute**-Objekt ist von einem [App-FeaturePython](App_FeaturePython/de.md)-Objekt abgeleitet und erbt alle seine Eigenschaften. Siehe [Assembly StarreVerbindungErstellen](Assembly_CreateJointFixed/de#Eigenschaften.md) für weitere Eigenschaften.
