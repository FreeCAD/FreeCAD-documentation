# OpenSCAD Hull/de


</div>

## Beschreibung

Wendet eine Hülle auf ausgewählte Formen an.

![Beispiel einer an zwei Zylindern durchgeführten Hüllen Operation](images/Hull_example.jpg )

Beispiel einer an zwei Zylindern durchgeführten Hüllen Operation

## Anwendung

1.  Formen in [Baumansicht](Tree_view/de.md) oder in [3D-Ansicht](3D_view/de.md) auswählen
2.  Klicke auf <img alt="" src=images/OpenSCAD_Hull.svg  style="width:32px;"> oder wähle **OpenSCAD**} → {**<img src="images/OpenSCAD_Hull.svg" width=32px> Hülle** aus dem oberen Menü.

## Begrenzungen

OpenSCAD muss installiert und konfiguriert sein in **Bearbeiten** → **Einstellungen**, um die Funktion nutzen zu können
