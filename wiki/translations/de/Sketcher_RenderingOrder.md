# Sketcher RenderingOrder/de



## Beschreibung

Der Befehl **Sketcher RenderingReihenfolge** selbst macht aktuell gar nichts. Aber im zugehörigen Menü kann die Rendering-Reihenfolge für alle Skizzen geändert werden.

Im Falle von deckungsgleichen Objekten gibt die Rendering-Reihenfolge vor, welche Elemente zuoberst liegen und daher in der [3D-Ansicht](3D_view/de.md) zuerst ausgewählt werden.



## Anwendung

1.  Auf den Nach-Unten-Pfeil rechts neben der Schaltfläche **<img src="images/Sketcher_RenderingOrder.svg" width=16px> [Rendering-Reihenfolge konfigurieren](Sketcher_RenderingOrder/de.md)** klicken.
2.  Die Liste mit der Rendering-Reihenfolge wird geöffnet:
    ![](images/Sketcher_RenderingOrder_Menu.png )
3.  Einzelne Listenelemente ziehen, um die Rendering-Reihenfolge zu ändern.
4.  Die (Farbe der) Schaltfläche wird passend zum ersten Element auf der Liste geändert.
5.  Ein Klick außerhalb der Liste schließt die Aktion ab.
