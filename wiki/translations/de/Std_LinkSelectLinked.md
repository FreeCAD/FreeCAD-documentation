# Std LinkSelectLinked/de



## Beschreibung

Der Befehl **Std ZumVerknüpftenObjektGehen** wählt die {{PropertyData/de|Linked Object}}, das Quellobjekt eines [App-Link](App_Link/de.md)-Objekts, eine Verknüpfung.



## Anwendung

1.  Ein Link-Objekt auswählen.
2.  Es gibt mehrere Möglichkeiten, den Befehl aufzurufen:
    -   Den Menüeintrag **Ansicht → Verknüpfungsnavigation → <img src="images/Std_LinkSelectLinked.svg" width=16px> Zum verknüpften Objekt gehen** auswählen.
    -   Die Menüoption **Verknüpfungen... → <img src="images/Std_LinkSelectLinked.svg" width=16px> Zum verknüpften Objekt gehen** im Kontextmenü der [Baumansicht](Tree_view/de.md) auswählen.
    -   Das Tastaturkürzel **S** dann **G**.
3.  Das verknüpfte Objekt wird ausgewählt. Wenn dieses Objekt zu einem externen Dokument gehört, ist jenes jetzt aktiviert.
4.  Optional kann <img alt="" src=images/Std_SelBack.svg  style="width:16px;"> [Std AuswahlZurück](Std_SelBack/de.md) verwendet werden, um das Link-Objekt erneut auszuwählen.
