# BIM TDView/de



## Beschreibung

Das Werkzeug <img alt="" src=images/BIM_TDView.svg  style="width:24px;"> [BIM TDAnsicht](BIM_TDView/de.md) erstellt eine neue TechDraw-Ansicht auf eine Schnittebene oder auf 2D-Objekte.

    -   Die Schaltfläche **<img src="images/BIM_TDView.svg" width=16px> [View](BIM_TDView.md)** drücken.
    -   Den Menüeintrag **Beschriftung → <img src="images/BIM_TDView.svg" width=16px> View** auswählen.
