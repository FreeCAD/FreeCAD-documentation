# Spreadsheet StyleItalic/de

## Beschreibung

Das **[<img src=images/Spreadsheet_StyleItalic.svg style="width:16px"> [Kalkulationstabelle StilKursiv](Spreadsheet_StyleItalic/de.md)** Werkzeug bewirkt, dass der Text innerhalb der Zelle den Stil **kursiv** erhält.

## Anwendung

1.  Stelle sicher, dass eine aktive **[<img src=images/Spreadsheet_CreateSheet.svg style="width:16px"> [Kalkulationstabelle](Spreadsheet_CreateSheet/de.md)** geöffnet ist, so dass die Schaltfläche aktiviert ist.
2.  Wähle eine Zelle.
3.  Drücke die **[<img src=images/Spreadsheet_StyleItalic.svg style="width:16px"> [Kalkulationstabelle StilKursiv](Spreadsheet_StyleItalic/de.md)** Schaltfläche.
