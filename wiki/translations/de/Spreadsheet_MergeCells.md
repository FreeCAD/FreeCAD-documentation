# Spreadsheet MergeCells/de

## Beschreibung

Das **[<img src=images/Spreadsheet_MergeCells.svg style="width:16px"> [Kalkulationstabelle ZellenZusammenführen](Spreadsheet_MergeCells/de.md)** Werkzeug führt mindestens zwei ausgewählte Zellen in einer bestehenden Kalkulationstabelle im aktiven Dokument zusammen.

## Anwendung

1.  Stelle sicher, dass eine aktive **[<img src=images/Spreadsheet_CreateSheet.svg style="width:16px"> [Kalkulationstabelle](Spreadsheet_CreateSheet/de.md)** geöffnet ist, so dass die Schaltfläche aktiviert ist.
2.  Wähle mindestens zwei aneinandergrenzende Zellen aus.
3.  Drücke die **[<img src=images/Spreadsheet_MergeCells.svg style="width:16px"> [Kalkulationstabelle ZellenZusammenführen](Spreadsheet_MergeCells/de.md)** Schaltfläche.
