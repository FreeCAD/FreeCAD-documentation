# CAM SelectLoop/de



## Beschreibung

Das Werkzeug **<img src="images/CAM_SelectLoop.svg" width=16px> [CAM SchleifeAuswählen](CAM_SelectLoop/de.md)** vervollständigt eine Auswahl von Kanten, die eine Schleife bilden.



## Anwendung

1.  Select an edge.
2.  There are several ways to invoke the command:
    -   Press the **<img src="images/CAM_SelectLoop.svg" width=16px> [CAM SelectLoop](CAM_SelectLoop.md)** button.
    -   Select the **CAM → <img src="images/CAM_SelectLoop.svg" width=16px> Finish Selecting Loop** option from the menu.
    -   Use the keyboard shortcut: **P** then **L**.
3.  The remaining edges of the loop are added to the selection.



## Hinweise

-   This command can also be very useful in other workbenches.
