# Std ViewRight/de



## Beschreibung

Der Befehl **Std SeitenansichtRechts** schwenkt die Kamerasicht der aktiven [3D-Ansicht](3D_view/de.md) in die Richtung der negativen X-Achse.

![](images/FreeCAD_views_front.svg ) 
*Der Pfeil 3 zeigt in Richtung der Seitenansicht von rechts*



## Anwendung

1.  Es gibt mehrere Möglichkeiten, den Befehl aufzurufen:
    -   Die Schaltfläche **<img src="images/Std_ViewRight.svg" width=16px> [Seitenansicht von rechts](Std_ViewRight/de.md)** drücken.
    -   Den Menüeintrag **Ansicht → Standardansichten → <img src="images/Std_ViewRight.svg" width=16px> Seitenansicht von rechts** auswählen.
    -   Die Menüoption **Standardansichten → <img src="images/Std_ViewRight.svg" width=16px> Seitenansicht von rechts** im Kontextmenü der [3D-Ansicht](3D_view/de.md) auswählen.
    -   Das Tastaturkürzel **3**.
