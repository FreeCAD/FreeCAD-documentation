# TechDraw ExtensionCreateHorizChainDimension/de



## Beschreibung

Das Werkzeug **TechDraw ErgänzungHorizontaleMaßkette** erstellt eine horizontale Maßkette: eine Folge von fluchtend ausgerichteten Maßen.

<img alt="" src=images/TechDraw_ExtensionCreateHorizChainDimensionExample.png  style="width:400px;"> 
*Rechts die erzeugten Maße*



## Anwendung

1.  Drei oder mehr Knotenpunkte auswählen.
2.  Die beiden linken Knotenpunkte bestimmen die Position der Maßkette.
3.  Es gibt mehrere Möglichkeiten, das Werkzeug aufzurufen:
    -   Die Schaltfläche **<img src="images/TechDraw_ExtensionCreateHorizChainDimension.svg" width=16px> [Horizontale Maßkette erstellen](TechDraw_ExtensionCreateHorizChainDimension/de.md)** drücken.
    -   Den Menüeintrag **TechDraw → Ergänzungen: Maße → <img src="images/TechDraw_ExtensionCreateHorizChainDimension.svg" width=16px> Horizontale Maßkette erstellen** auswählen.
4.  Eine Maßkette mit mittigen Maßzahlen wird erstellt.
