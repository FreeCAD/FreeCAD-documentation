# Raytracing InsertPart/de



## Beschreibung

Fügt eine Ansicht eines Teils in ein Raytracing-Projekt ein.



## Anwendung

1.  Wähle das Objekt in der [Baumansicht](Tree_view/de.md) aus.
2.  Klicke auf <img alt="" src=images/Raytracing_InsertPart.svg  style="width:32px;"> oder wähle ** Raytracing** → **<img src="images/Raytracing_InsertPart.svg" width=32px> Teil einfügen** aus dem Menü.
3.  Die Ansicht des ausgewählten Teils wird in das Raytracing-Projekt eingeführt.
4.  Falls mehr als ein Raytracing-Projekt im aktiven Dokument existiert, muss eins ausgewählt werden.
