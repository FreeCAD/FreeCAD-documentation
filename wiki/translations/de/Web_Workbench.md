# <img alt="Web Arbeitsbereichssymbol" src=images/Workbench_Web.svg  style="width:64px;"> Web Workbench/de

## Einleitung

Der [Web-Arbeitsbereich](Web_Workbench/de.md) kein richtiger Modellierungsarbeitsbereich, sondern stellt innerhalb von FreeCAD ein Browser-Fenster zur Verfügung.

Von diesem Browser aus kann man im Web navigieren, auf Verweise (Links) klicken, die Online-Dokumentation lesen und sich an Webseiten anmelden (Cookies werden unterstützt). Der Arbeitsbereich Web ist auch eine Basis zur Entwicklung von Werkzeugen mit Web-Bezug in FreeCAD. Das Browser-Fenster basiert auf der [QtWebEngine](https://wiki.qt.io/QtWebEngine) und hält Standard-Navigationsschaltflächen wie Ladevorgang anhalten, Webseite aktualisieren, Vorherige Seite, Nächste Seite, Vergrößern und Verkleinern vor.

## Werkzeuge

Der Arbeitsbereich Web enthält folgende Werkzeuge:

-   <img alt="" src=images/Web_BrowserSetURL.svg  style="width:32px;"> [URL bearbeiten](Web_BrowserSetURL/de.md): Ändert die URL des aktiven Web-Reiters.

-   <img alt="" src=images/Web_OpenWebsite.svg  style="width:32px;"> [Webseite öffnen\...](Web_OpenWebsite/de.md): Öffnet die [FreeCAD-Webseite](https://freecadweb.org).

-   <img alt="" src=images/Web_BrowserBack.svg  style="width:32px;"> [Vorherige Seite](Web_BrowserBack/de.md): Ruft die vorherige Webseite auf.

-   <img alt="" src=images/Web_BrowserNext.svg  style="width:32px;"> [Nächste Seite](Web_BrowserNext/de.md): Ruft die nächste Webseite auf.

-   <img alt="" src=images/Web_BrowserRefresh.svg  style="width:32px;"> [Webseite aktualisieren](Web_BrowserRefresh/de.md): Aktualisiert die aktive Webseite.

-   <img alt="" src=images/Web_BrowserStop.svg  style="width:32px;"> [Ladevorgang anhalten](Web_BrowserStop/de.md): Stoppt das Laden der aktiven Webseite.

-   <img alt="" src=images/Web_BrowserZoomIn.svg  style="width:32px;"> [Vergrößern](Web_BrowserZoomIn/de.md): Vergrößert die aktive Webseite.

-   <img alt="" src=images/Web_BrowserZoomOut.svg  style="width:32px;"> [Verkleinern](Web_BrowserZoomOut/de.md): Verkleinert die aktive Webseite.
