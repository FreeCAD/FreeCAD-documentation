# Spreadsheet Import/en

## Description

The <img alt="" src=images/Spreadsheet_Import.svg  style="width:24px;"> **Spreadsheet Import** tool imports a CSV file into a new spreadsheet. Note that the {{Value|tab}} character, and not the {{Value|comma}}, is the default field delimiter. But this can be changed in the [Spreadsheet Preferences](Spreadsheet_Preferences.md) (<small>(v0.20)</small> ).

## Usage

1.  There are several ways to invoke the tool:
    -   Press the **[<img src=images/Spreadsheet_Import.svg style="width:16px"> [Import spreadsheet](Spreadsheet_Import.md)** button.
    -   Select the **Spreadsheet → [<img src=images/Spreadsheet_Import.svg style="width:16px"> Import spreadsheet** option from the menu.
2.  Select a CSV file.
3.  Press the **Open** button.

## Notes

-   Use the [Std Import](Std_Import.md) command to import Excel ***.xlsx** files.
