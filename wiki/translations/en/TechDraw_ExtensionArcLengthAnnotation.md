# TechDraw ExtensionArcLengthAnnotation/en

## Description

The **TechDraw ExtensionArcLengthAnnotation** tool calculates the arc length of selected edges and inserts an arc length annotation.

<img alt="" src=images/TechDraw_ExtensionArcLengthAnnotationExample.png  style="width:480px;"> 
*The calculated length of a B-spline approximation of sin(x) from 0 to 2π*

## Usage

1.  Select one or more edges.
2.  There are several ways to invoke the tool:
    -   Press the **<img src="images/TechDraw_ExtensionArcLengthAnnotation.svg" width=16px> [Calculate the arc length of selected edges](TechDraw_ExtensionArcLengthAnnotation.md)** button.
    -   Select the **TechDraw → Extensions: Attributes/Modifications → <img src="images/TechDraw_ExtensionArcLengthAnnotation.svg" width=16px> Calculate the arc length of selected edges** option from the menu.
3.  The total arc length of the edges is calculated and an annotation is inserted.
