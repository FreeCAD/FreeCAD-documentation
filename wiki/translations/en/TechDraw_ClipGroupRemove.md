# TechDraw ClipGroupRemove/en

## Description

The **TechDraw ClipGroupRemove** tool removes a View from an existing Clip and returns it to the Page area.

![](images/TechDraw_Clipview.png ) 
*Viewport window clipping different existing views*

## Usage

1.  Select a View that is part of a Clip in the [3D view](3D_view.md) or [Tree view](Tree_view.md).
2.  There are several ways to invoke the tool:
    -   Press the **<img src="images/TechDraw_ClipGroupRemove.svg" width=16px> [Remove View from Clip Group](TechDraw_ClipGroupRemove.md)** button.
    -   Select the **TechDraw → Clipped Views → <img src="images/TechDraw_ClipGroupRemove.svg" width=16px> Remove View from Clip Group** option from the menu.
