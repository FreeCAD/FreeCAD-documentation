# CAM Inspect/en

## Description

The <img alt="" src=images/CAM_Inspect.svg  style="width:24px;"> [Inspect](CAM_Inspect.md) tool allows inspection of the internal FreeCAD G-code dialect contents of a CAM Operation object.

![](images/Path_inspector.jpg )

## Usage

1.  Select a CAM object.
2.  There are several ways to invoke the command:
    -   Press the **<img src="images/CAM_Inspect.svg" width=16px> [Inspect CAM Commands](CAM_Inspect.md)** button.
    -   Select the **CAM → <img src="images/CAM_Inspect.svg" width=16px> Inspect CAM Commands** option from the menu.
    -   Select the **<img src="images/CAM_Inspect.svg" width=16px> Inspect G-code** option from the [Tree view](Tree_view.md) context menu.
    -   Use the keyboard shortcut: **P** then **I**.

## Options

-   The font and colors of the editor window follow the settings from **Edit → Preferences → Editor**
