# Assembly CreateJointPerpendicular/en

## Description

The <img alt="" src=images/Assembly_CreateJointPerpendicular.svg  style="width:24px;"> [Assembly CreateJointPerpendicular](Assembly_CreateJointPerpendicular.md) tool creates a Perpendicular joint between two selected parts, setting the Z axes of selected coordinate systems perpendicular.

## Usage

1.  There are several ways to invoke the tool:
    -   Press the **<img src="images/Assembly_CreateJointPerpendicular.svg" width=16px> [Create Perpendicular Joint](Assembly_CreateJointPerpendicular.md)** button.
    -   Select the **Assembly → <img src="images/Assembly_CreateJointPerpendicular.svg" width=16px> Create Perpendicular Joint** option from the menu.
    -   Use the keyboard shortcut: **M**.
