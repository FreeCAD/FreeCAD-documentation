# Spreadsheet AlignTop/en

## Description

The <img alt="" src=images/Spreadsheet_AlignTop.svg  style="width:24px;"> **Spreadsheet AlignTop** tool aligns the contents of selected cells to the top.

## Usage

1.  Make sure there is an active [spreadsheet](Spreadsheet_CreateSheet.md).
2.  Select one or more cells.
3.  There are several ways to invoke the tool:
    -   Press the **[<img src=images/Spreadsheet_AlignTop.svg style="width:16px"> [Align top](Spreadsheet_AlignTop.md)** button.
    -   Select the **Spreadsheet → Alignment → [<img src=images/Spreadsheet_AlignTop.svg style="width:16px"> Align top** option from the menu.
