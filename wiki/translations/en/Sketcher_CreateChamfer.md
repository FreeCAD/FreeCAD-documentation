# Sketcher CreateChamfer/en

## Description

The <img alt="" src=images/Sketcher_CreateChamfer.svg  style="width:24px;"> [Sketcher CreateChamfer](Sketcher_CreateChamfer.md) tool creates a chamfer between two non-parallel edges. This is the same tool as [Sketcher CreateFillet](Sketcher_CreateFillet.md) but with a different initial mode.

## Usage

See also: [Drawing aids](Sketcher_Workbench#Drawing_aids.md).

1.  There are several ways to invoke the tool:
    -   Press the **<img src="images/Sketcher_CreateChamfer.svg" width=16px> [Create chamfer](Sketcher_CreateChamfer.md)** button.
    -   Select the **Sketcher → Sketcher tools → <img src="images/Sketcher_CreateChamfer.svg" width=16px> Create chamfer** option from the menu.
    -   Right-click in the [3D view](3D_view.md) and select the **<img src="images/Sketcher_CreateChamfer.svg" width=16px> Create chamfer** option from the context menu.
    -   Use the keyboard shortcut: **G** then **F**, then **C**.
2.  For further steps see [Sketcher CreateFillet](Sketcher_CreateFillet#Usage.md).
