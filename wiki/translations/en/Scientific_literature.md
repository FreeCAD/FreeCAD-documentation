# Scientific literature/en
## Description

This page collects some articles published that reference or use the FreeCAD system.

Most of these articles are open-access.

-   \"Parametric CAD modeling for open source scientific hardware: Comparing OpenSCAD and FreeCAD Python scripts\", Felipe Machado, Norberto Malpica, Susana Borromeo, [DOI: 10.1371/journal.pone.0225795](https://journals.plos.org/plosone/article?id=10.1371/journal.pone.0225795).
-   \"Low-cost 4D BIM modelling: a comparison between FreeCAD and commercial software\", D. Di Donato, Matteo Abita, [DOI: 10.5194/isprs-archives-XLII-2-W17-107-2019](https://www.int-arch-photogramm-remote-sens-spatial-inf-sci.net/XLII-2-W17/107/2019/).
-   \"Interactive Programming for Parametric CAD\", Aman Mathur, Marcus Pirron, Damien Zufferey, [DOI: 10.1111/cgf.14046](https://doi.org/10.1111/cgf.14046).
