# Sketcher ConstrainVertical/sv


</div>



## Beskrivning

The <img alt="" src=images/Sketcher_ConstrainVertical.svg  style="width:24px;"> [Sketcher ConstrainVertical](Sketcher_ConstrainVertical.md) tool constrains lines or pairs of points to be vertical (parallel to the vertical axis of the sketch).


<small>(v1.0)</small> 

: In most cases it is advisable to use the combined [Sketcher ConstrainHorVer](Sketcher_ConstrainHorVer.md) tool instead.

### [Continue mode](Sketcher_Workbench#Continue_modes.md) 

1.  Make sure there is no selection.
2.  There are several ways to invoke the tool:
    -   
        <small>(v1.0)</small> 
        
        : If the **Auto tool for Horizontal/Vertical** [preference](Sketcher_Preferences#General.md) is selected (default): press the down arrow to the right of the **<img src="images/Sketcher_ConstrainHorVer.svg" width=|x16px><img src="images/Toolbar_flyout_arrow.svg" width=x16px>** button and select the **<img src="images/Sketcher_ConstrainVertical.svg" width=16px> Constrain vertical** option from the dropdown.

    -   If this preference is not selected (and in {{VersionMinus|0.21}}): press the **<img src="images/Sketcher_ConstrainVertical.svg" width=16px> [Constrain vertical](Sketcher_ConstrainVertical.md)** button.

    -   Select the **Sketch → Sketcher constraints → <img src="images/Sketcher_ConstrainVertical.svg" width=16px> Constrain vertical** option from the menu.

    -   Use the keyboard shortcut: **V**.
3.  For further steps see [Sketcher ConstrainHorizontal](Sketcher_ConstrainHorizontal#Continue_mode.md).
