# Part MakeSolid/ro


</div>

## Description


<div class="mw-translate-fuzzy">

## Introducere

Această comandă creează un solid dintr-un obiect tip formă.


</div>

This command is typically used as one of the steps to create a solid from a mesh. See [Part ShapeFromMesh](Part_ShapeFromMesh#Usage.md) for more information.

## Usage


<div class="mw-translate-fuzzy">

## Utilizare

1.  Selectați forma obiect.
2.  Selectați ** Part** → **Convert to solid** din meniul principal.
3.  Un nou obiect solid este creat din obiectul tip formă.

.


</div>

## Notes


<div class="mw-translate-fuzzy">

## Limitări

Nu va fi nici o analiză sau validare a obiectului de formă.
Este recomandabil să utilizați traseul ** Part** → ** Refine Shape** înaintea conversiei în solid.


</div>

## Properties

See also: [Property editor](Property_editor.md).

The created objects are [Part Feature](Part_Feature.md) objects with no additional properties.
