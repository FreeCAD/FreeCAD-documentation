În FreeCAD, Plasamentul definește poziția și rotația unui obiect. Conceptul de Plasament este explicat în detaliu aici: [Placement](Placement.md).

Exemplu de setare a destinației de plasare a obiectului de document: 
```python
myObj = FreeCAD.ActiveDocument.ActiveObject
pl = FreeCAD.Placement()
pl.move(FreeCAD.Vector(2,0,0))
myObj.Placement = pl
```
