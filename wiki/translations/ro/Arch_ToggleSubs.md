## Descriere


<div class="mw-translate-fuzzy">

Aceste instrumente comută vizibilitatea tuturor subtracțiilor unui obiect [ Arch](Arch_Workbench/ro.md) între vizibil și ascuns. Dacă un obiect Arch conține unele obiecte vizibile și unele obiecte ascunse substrase, toate vor deveni vizibile / ascunse după executarea acestei comenzi.


</div>

Normally, if an Arch object, like an [Arch Wall](Arch_Wall.md), is selected and you press **Space** only the external wall will be hidden or made visible, but not the internal objects.

With this tool, the internal subtracted objects will all become visible or hidden.




<div class="mw-translate-fuzzy">

## Cum se folosește 


</div>


<div class="mw-translate-fuzzy">

1.  Selectați un obiect care a subtracții
2.  Selectați meniul Arch → Utilities → **<img src="images/Arch_ToggleSubs‏‎.svg" width=16px> [Toggle Subcomponents‏‎](Arch_ToggleSubs‏‎.md)
**


</div>



## Opţiuni


<div class="mw-translate-fuzzy">

Opțiunea preferinței **Edit -\> Preferences → Arch → Apply Construction style** opțiunea Preferințe permite să dați o culoare tuturor obiectelor extrase/substrase, culoarea dată de setările Preferences → Draft → General Draft, dă o performanță mai bună acestei comenzi.


</div>

The [Draft construction](Draft_ToggleConstructionMode.md) style can be set in the [Draft Preferences](Draft_Preferences.md), in the menu **Preferences → Draft → General Draft**.
