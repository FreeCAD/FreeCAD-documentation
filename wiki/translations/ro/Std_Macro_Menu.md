## Tools

The following tools are available in this menu:


<div class="mw-translate-fuzzy">

[Macro recording](Std_DlgMacroRecord.md) deschide un dialog pentru înregistrarea macrocomenzii


</div>


<div class="mw-translate-fuzzy">

[Macros\...](Std_DlgMacroExecute.md) selectează și execută macrocomanda înregistrată.


</div>

-   Recent macros: shows a list of recently used macros, which can be called with a key combination, for example, **Ctrl**+**Shift**+**1**. This can be configured in the [Preferences](Preferences_Editor.md).


<div class="mw-translate-fuzzy">

[Execute macro](Std_DlgMacroExecuteDirect.md) execută macrocomanda curentă din editorul de macrocomenzi.


</div>

-   [Attach to remote debugger\...](Std_MacroAttachDebugger.md): Attaches FreeCAD to a remote debugger.


<div class="mw-translate-fuzzy">

[Debug macro](Std_MacroStartDebug.md) pornește sesiunea de depanarea pentru macrocomanda curentă aflată în editorul de macrocomenzi


</div>


<div class="mw-translate-fuzzy">

[Stop debugging](Std_MacroStopDebug.md) încheie sesiunea de depanare.


</div>


<div class="mw-translate-fuzzy">

</div>
