# Std WhatsThis/ro


</div>



## Descriere


<div class="mw-translate-fuzzy">

Acest instrument oferă informații despre o comandă selectată prin deschiderea modului offline. [Help documentation](Std_OnlineHelp.md) în pagina corespunzătoare.


</div>

Since FreeCAD version 0.20 this command and the Help system are managed by a separate Help Module. For versions 0.20 and 0.21 this module is an [addon](https://github.com/FreeCAD/FreeCAD-Help) that must be [installed](Std_AddonMgr.md) separately. In later versions the Help Module is integrated in FreeCAD.




<div class="mw-translate-fuzzy">

## Cum se folosește 


</div>


<div class="mw-translate-fuzzy">

1.  Apăsați butonul **[<img src=images/Std_WhatsThis.png style="width:24px"> '''What's This?'''** .
2.  Faceți clic pe o pictogramă a barei de instrumente sau pe o intrare din meniu.


</div>
