# Spreadsheet Import/fr

## Description

L\'outil <img alt="" src=images/Spreadsheet_Import.svg  style="width:24px;"> **Spreadsheet Importer** importe un fichier CSV dans une nouvelle feuille de calcul. Notez que le caractère {{Value|tab}}, et non {{Value|comma}}, est le délimiteur de champ par défaut. Cela peut être modifié dans les [Spreadsheet Préférences](Spreadsheet_Preferences/fr.md). ({{Version/fr|0.20}}).



## Utilisation

1.  Il y a plusieurs façons de lancer l\'outil :
    -   Appuyez sur le bouton **[<img src=images/Spreadsheet_Import.svg style="width:16px"> [Importer une feuille de calcul](Spreadsheet_Import/fr.md)**.
    -   Sélectionnez l\'option **Spreadsheet → [<img src=images/Spreadsheet_Import.svg style="width:16px"> Importer une feuille de calcul** du menu.
2.  Sélectionnez un fichier CSV.
3.  Appuyez sur le bouton **Ouvrir**.



## Remarques

-   Utilisez la commande [Std Importer](Std_Import/fr.md) pour importer des fichiers Excel ***.xlsx**.
