# Std FreeCADPowerUserHub/fr

## Description

La commande **Std Documentation pour programmer en Python** ouvre la page de documentation pour utilisateurs expérimentés dans le navigateur Internet par défaut du système. La commande nécessite une connexion Internet fonctionnelle.

La commande pointe vers [<https://wiki.freecad.org/Power_users_hub>](https://wiki.freecad.org/Power_users_hub).



## Utilisation

-   Sélectionnez l\'option **Aide → [<img src=images/Std_FreeCADPowerUserHub.svg style="width:16px"> [Documentation pour programmer en Python](Std_FreeCADPowerUserHub/fr.md)** du menu.
