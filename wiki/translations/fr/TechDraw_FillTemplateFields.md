# TechDraw FillTemplateFields/fr

## Description

L\'outil <img alt="" src=images/TechDraw_FillTemplateFields.svg  style="width:24px;">[Remplir les champs du modèle](TechDraw_FillTemplateFields/fr.md) remplit les champs de texte modifiables dans le bloc titre de la page en cours.



## Utilisation

1.  Il y a plusieurs façons de lancer l\'outil :
    -   Appuyez sur le **<img src="images/TechDraw_FillTemplateFields.svg" width=16px> [Mettre à jour les champs du modèle](TechDraw_FillTemplateFields/fr.md)**.
    -   Sélectionnez l\'option **TechDraw → Page → <img src="images/TechDraw_FillTemplateFields.svg" width=16px> Mettre à jour les champs du modèle** du menu.
2.  \...



## Remarques

Cet outil semble n\'avoir aucun rapport avec le nouvel attribut **freecad:autofill** de l\'espace de noms SVG de FreeCAD qui est apparu à peu près au même moment.
