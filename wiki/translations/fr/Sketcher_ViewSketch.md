# Sketcher ViewSketch/fr

## Description

Cet outil définit la vue du modèle perpendiculairement au plan de l\'esquisse. Il est utile lorsque l\'utilisateur a modifié l\'orientation de la vue du modèle pour examiner un autre aspect du modèle et qu\'il souhaite revenir à une vue normale par rapport à l\'esquisse.



## Utilisation

En mode d\'édition d\'esquisse, il existe plusieurs façons de lancer cette commande :

-   Cliquez sur l\'icône **[<img src=images/Sketcher_ViewSketch.svg style="width:16px"> [Vue de l'esquisse](Sketcher_ViewSketch/fr.md)** dans la barre d\'outils Sketcher ou Part Design
-   Allez dans le menu **Esquisse → [<img src=images/Sketcher_ViewSketch.svg style="width:16px"> Vue d'esquisse**.
