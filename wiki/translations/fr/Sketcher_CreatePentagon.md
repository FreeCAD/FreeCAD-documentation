# Sketcher CreatePentagon/fr

## Description

L\'outil <img alt="" src=images/Sketcher_CreatePentagon.svg  style="width:24px;"> [Sketcher Pentagone](Sketcher_CreatePentagon/fr.md) crée un pentagone. {{Version/fr|1.0}} : il s\'agit du même outil que [Sketcher Polygone régulier](Sketcher_CreateRegularPolygon/fr.md) mais avec un nombre de côtés prédéfini à 7.

![](images/SketcherCreatePentagonExample.png‎ )



## Utilisation

Voir aussi : [Aides au dessin](Sketcher_Workbench/fr#Aides_au_dessin.md).

1.  Il y a plusieurs façons de lancer l\'outil :
    -   Appuyez sur le bouton **<img src="images/Sketcher_CreatePentagon.svg" width=16px> [Créer un pentagone](Sketcher_CreatePentagon/fr.md)**.
    -   Sélectionnez l\'option **Esquisse → Géométries d'esquisse → <img src="images/Sketcher_CreatePentagon.svg" width=16px> Créer un pentagone** du menu.
    -   Utilisez le raccourci clavier : **G** puis **P**, puis **5**.
2.  Pour d\'autres étapes, voir [Sketcher Polygone régulier](Sketcher_CreateRegularPolygon/fr#Utilisation.md).
