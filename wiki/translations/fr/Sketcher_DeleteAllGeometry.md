# Sketcher DeleteAllGeometry/fr

## Description

Cet outil supprime tous les éléments de géométrie de l\'esquisse en cours d\'édition.



## Utilisation

1.  Allez dans le menu **Esquisse → Outils d'esquisse → [<img src=images/Sketcher_DeleteAllGeometry.svg style="width:16px"> Supprimer tous les éléments de géométrie**.
2.  Répondez **Oui** au message affiché.
