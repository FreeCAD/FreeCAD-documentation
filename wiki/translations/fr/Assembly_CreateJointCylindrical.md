# Assembly CreateJointCylindrical/fr

## Description

L\'outil <img alt="" src=images/Assembly_CreateJointCylindrical.svg  style="width:24px;"> [Assemblage Liaison pivot glissant](Assembly_CreateJointCylindrical/fr.md) crée une liaison pivot glissant entre deux pièces sélectionnées, permettant une rotation autour d\'un seul axe et un mouvement le long du même axe.



## Utilisation

1.  Vous pouvez sélectionner deux entités géométriques de deux formes différentes. Les autres sélections seront rejetées.
2.  Il y a plusieurs façons de lancer l\'outil :
    -   Appuyez sur le bouton **<img src="images/Assembly_CreateJointCylindrical.svg" width=16px> [Créer une liaison pivot glissant](Assembly_CreateJointCylindrical/fr.md)**.
    -   Sélectionnez la **Assemblage → <img src="images/Assembly_CreateJointCylindrical.svg" width=16px> Créer une liaison pivot glissant** du menu.
    -   Utilisez le raccourci clavier : **C**.
3.  La fenêtre de dialogue **Créer une liaison** s\'ouvre dans le [panneau des tâches](Task_panel/fr.md) avec la liste des entités présélectionnées.
4.  Pour les étapes suivantes, voir [Assemblage Liaison fixe](Assembly_CreateJointFixed/fr#Utilisation.md).



## Propriétés

Voir aussi : [Éditeur de propriétés](Property_editor/fr.md)

Un objet **Cylindrical** est dérivé d\'un objet [App FeaturePython](App_FeaturePython/fr.md) et hérite de toutes ses propriétés. Voir [Assembly Liaison fixe](Assembly_CreateJointFixed/fr#Propriétés.md) pour des propriétés supplémentaires.
