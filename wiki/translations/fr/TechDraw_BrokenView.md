# TechDraw BrokenView/fr



## Description

L\'outil **TechDraw Vue interrompue** insère une \"vue interrompue\" qui est soit basée sur une [vue d\'une pièce](TechDraw_View/fr.md) existante, soit sur un ou plusieurs objets, tels que des [corps](PartDesign_Body/fr.md) ou des [Parts](Std_Part/fr.md). La vue interrompue nécessite également une ou plusieurs [esquisses](Sketcher_NewSketch/fr.md) qui définissent l\'emplacement et la taille des zones à supprimer. La vue interrompue se comporte de la même manière que les autres vues. La direction de projection est tirée de la vue de la pièce existante, de la direction de la caméra 3D ou de la normale d\'une face sélectionnée.

## Utilisation

1.  Vous pouvez faire pivoter la [vue 3D](3D_view/fr.md). La vue 3D détermine les valeurs initiales des propriétés **Direction** et **XDirection** de la vue interrompue.
2.  Sélectionnez l\'objet pour lequel vous souhaitez créer une vue interrompue ou sélectionnez une vue TechDraw existante qui contient cet objet.
3.  Ajoutez une ou plusieurs esquisses de rupture à la sélection en les sélectionnant dans la [vue en arborescence](Tree_view/fr.md). Chaque esquisse ne doit contenir que deux lignes parallèles. Vous pouvez également utiliser d\'autres objets comportant deux bords parallèles.
4.  Il existe plusieurs façons de lancer l\'outil :
    -   Appuyez sur le **<img src="images/TechDraw_BrokenView.svg" width=16px> [Insérer une vue interrompue](TechDraw_BrokenView/fr.md)**.
    -   Sélectionnez l\'option **TechDraw → Vues de Techdraw → <img src="images/TechDraw_BrokenView.svg" width=16px> Insérer une vue interrompue** du menu.



## Propriétés

Voir aussi : [Éditeur de propriétés](Property_editor/fr.md)

Une vue interrompue, en fait un objet {{Incode|TechDraw::DrawBrokenView}}, est dérivée d\'une [vue de Part](TechDraw_View/fr#Propriétés_Vue_de_Part.md) et hérite de toutes ses propriétés. Elle possède également les propriétés supplémentaires suivantes :



### Données


{{TitleProperty|Broken View}}

-    **Breaks|LinkList**: objets de la vue 3D qui définissent les points de départ et d\'arrivée ainsi que la direction des cassures dans cette vue.

-    **Gap|Length**: distance de séparation entre les cassures dans cette vue (longueur 3D non mise à l\'échelle).



## Remarques

-   Les cassures doivent être verticales ou horizontales. Les cassures inclinées ne sont pas prises en charge.
-   Voir aussi [TechDraw Vue](TechDraw_View/fr.md).



## Script

Voir aussi : [Autogenerated API documentation](https://freecad.github.io/SourceDoc/) et [FreeCAD Débuter avec les scripts](FreeCAD_Scripting_Basics/fr.md).

Une Vue interrompue peut être créée à partir de [macros](Macros/fr.md) et de la console [Python](Python/fr.md) en utilisant la fonction suivante :


```python
doc = FreeCAD.ActiveDocument
box = doc.Box
profile = doc.Sketch
page = doc.Page

brokenView = doc.addObject("TechDraw::DrawBrokenView", "BrokenView")
page.addView(brokenView)
brokenView.Source= box
brokenView.Breaks = [doc.Sketch]
```
