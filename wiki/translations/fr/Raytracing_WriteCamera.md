# Raytracing WriteCamera/fr

## Description

Exporte les propriétés actuelles de la caméra dans un fichier pov à utiliser dans un projet POV-Ray.

## Utilisation

1.  Assurez-vous que la vue est définie sur [Vue en perspective](Std_PerspectiveCamera/fr.md)
2.  Choisissez **Raytracing** → **Utilitaires** →**<img src="images/Raytracing_WriteCamera.svg" width=32px> Exportez la caméra vers POV-Ray...** dans le menu principal.
