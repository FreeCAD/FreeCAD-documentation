# Assembly CreateJointRevolute/fr

## Description

L\'outil <img alt="" src=images/Assembly_CreateJointRevolute.svg  style="width:24px;"> [Assembly Liaison pivot](Assembly_CreateJointRevolute/fr.md) crée une liaison pivot (également connu sous le nom de liaison articulé, ou charnière en abrégé) permettant la rotation autour d\'un seul axe entre deux pièces sélectionnées.



## Utilisation

1.  Vous pouvez sélectionner deux entités géométriques de deux formes différentes. Les autres sélections seront rejetées.
2.  Il y a plusieurs façons de lancer l\'outil :
    -   Appuyez sur le bouton **<img src="images/Assembly_CreateJointRevolute.svg" width=16px> [Créer une liaison pivot](Assembly_CreateJointRevolute/fr.md)
**
    -   Sélectionnez l\'option **Assemblage → <img src="images/Assembly_CreateJointRevolute.svg" width=16px> Créer une liaison pivot** du menu.
    -   Utilisez le raccourci clavier : **R**.
3.  La fenêtre de dialogue **Créer une liaison** s\'ouvre dans le [panneau des tâches](Task_panel/fr.md) avec la liste des entités présélectionnées.
4.  Pour les étapes suivantes, voir [Assemblage Liaison fixe](Assembly_CreateJointFixed/fr#Utilisation.md).



## Propriétés

Voir aussi : [Éditeur de propriétés](Property_editor/fr.md)

Un objet **Revolute** est dérivé d\'un objet [App FeaturePython](App_FeaturePython/fr.md) et hérite de toutes ses propriétés. Voir [Assembly Liaison fixe](Assembly_CreateJointFixed/fr#Propriétés.md) pour des propriétés supplémentaires.
