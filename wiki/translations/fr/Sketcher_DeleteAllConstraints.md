# Sketcher DeleteAllConstraints/fr

## Description

L\'outil <img alt="" src=images/Sketcher_DeleteAllConstraints.svg  style="width:24px;"> [Sketcher Supprimer toutes les contraintes](Sketcher_DeleteAllConstraints/fr.md) supprime toutes les contraintes de l\'esquisse.



## Utilisation

1.  Il y a plusieurs façons de lancer l\'outil :
    -   Sélectionnez l\'option **Esquisse → Outils d'esquisse → [<img src=images/Sketcher_DeleteAllConstraints.svg style="width:16px"> Supprimer toutes les contraintes** du menu.

    -   
        {{Version/fr|1.0}}
        
        : cliquez avec le bouton droit de la souris dans la [vue 3D](3D_view/fr.md) et sélectionnez l\'option **<img src="images/Sketcher_DeleteAllConstraints.svg" width=16px> Supprimer toutes les contraintes** du menu contextuel (non disponible s\'il y a une sélection).
2.  La fenêtre de dialogue **Supprimer toutes les contraintes** s\'ouvre.
3.  Appuyez sur le bouton **Oui** pour confirmer et arrêter l\'outil.
