# Sketcher SelectConflictingConstraints/fr

## Description

Sélectionne les contraintes conflictuelles d\'une esquisse.

## Utilisation

1.  Entrez en mode édition de l\'esquisse.
2.  Choisissez **Esquisse → Outils d'esquisse → [<img src=images/Sketcher_SelectConflictingConstraints.svg style="width:16px"> Sélectionner les contraintes conflictuelles**.
3.  Les contraintes conflictuelles de l\'esquisse seront sélectionnées.
