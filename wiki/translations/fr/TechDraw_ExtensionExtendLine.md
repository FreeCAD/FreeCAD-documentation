# TechDraw ExtensionExtendLine/fr

## Description

L\'outil **TechDraw Prolonger** prolonge une ligne cosmétique ou une ligne centrale à ses deux extrémités.

<img alt="" src=images/TechDraw_ExtensionExtendLineExample.png  style="width:300px;"> 
*À droite, une ligne centrale prolongée aux deux extrémités*



## Utilisation

1.  Spécifiez à votre convenance la distance delta avec l\'outil <img alt="" src=images/TechDraw_ExtensionSelectLineAttributes.svg  style="width:16px;">. [TechDraw Sélection des attributs](TechDraw_ExtensionSelectLineAttributes/fr.md).
2.  Sélectionnez une seule ligne cosmétique ou une ligne centrale.
3.  Il existe plusieurs façons de lancer l\'outil :
    -   Appuyez sur le bouton **<img src="images/TechDraw_ExtensionExtendLine.svg" width=16px> [Prolonger la ligne](TechDraw_ExtensionExtendLine/fr.md)**.
    -   Sélectionnez l\'option **TechDraw → Extensions : attributs/modifications → <img src="images/TechDraw_ExtensionExtendLine.svg" width=16px> Prolonger la ligne** à partir du menu.
4.  La ligne est prolongée à ses deux extrémités.
