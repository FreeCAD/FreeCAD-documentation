## Description

Utilisez cette macro pour envoyer un ping à un autre utilisateur du forum afin que celui-ci reçoive une notification. Pour effectuer un ping, il faut connaître le nom d\'utilisateur exact et le numéro d\'identification de l\'utilisateur. Vous devez également connaître la syntaxe requise pour produire le ping. Le moins que l\'on puisse dire, c\'est qu\'elle est très peu conviviale. La macro Pinger est là pour vous sauver la mise. Exécutez la macro, sélectionnez l\'utilisateur que vous souhaitez pinger, cliquez sur le bouton **Ping**, copiez-collez le code dans le message du forum.

<img alt="" src=images/Pinger_scr.png  style="width:600px;"> 
*Capture d'écran de Pinger *

## Paramètres

La macro prend en charge les paramètres utilisateur, qui sont utilisés pour la liste des utilisateurs ajoutés. Ils se trouvent dans User parameter:Plugins/Pinger sous le nom de \"AddedUsers\", une chaîne contenant la représentation d\'un dictionnaire Python.
