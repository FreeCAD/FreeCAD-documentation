# Sketcher CreatePeriodicBSplineByInterpolation/fr

## Description

Cet outil crée une courbe B-spline périodique passant par des points donnés. Voir [cette page](B-Splines/fr.md) pour plus d\'informations sur les B-splines.



## Utilisation

1.  Appuyez sur le bouton **[<img src=images/Sketcher_CreatePeriodicBSplineByInterpolation.svg style="width:16px"> [B-spline périodique par des noeuds](Sketcher_CreatePeriodicBSplineByInterpolation/fr.md)**.
2.  Pour les autres étapes, voir [Sketcher B-spline par des nœuds](Sketcher_CreateBSplineByInterpolation/fr#Utilisation.md).



## Remarques

Voir [Sketcher B-spline simple](Sketcher_CreateBSpline/fr#Remarques.md).

## Limitations
