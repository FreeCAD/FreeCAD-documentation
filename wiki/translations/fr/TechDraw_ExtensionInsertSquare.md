# TechDraw ExtensionInsertSquare/fr

## Description

L\'outil **TechDraw Section tube** insère un symbole \"□\" au début du texte de la cote.

<img alt="" src=images/TechDraw_extensionInsertSquareExample.png  style="width:500px;"> 
*A droite, les cotes avec le symbole inséré*



## Utilisation

1.  Sélectionnez une ou plusieurs cotes.
2.  Il existe plusieurs façons de lancer l\'outil :
    -   Appuyez sur le bouton **<img src="images/TechDraw_ExtensionInsertSquare.svg" width=16px> [Insérer le préfixe "□"](TechDraw_ExtensionInsertSquare/fr.md)**.
    -   Sélectionnez la commande **TechDraw → Extensions : cotes → <img src="images/TechDraw_ExtensionInsertSquare.svg" width=16px> Insérer un préfixe "□"** à partir du menu.
3.  Le symbole \"□\" est inséré dans le texte de cote des cotes sélectionnées.
