# Raytracing WritePart/fr


{{Raytracing_Tools_navi/fr}}

Exporte la pièce sélectionnée dans un fichier pov à utiliser dans un projet POV-Ray.


{{Userdocnavi/fr}}

1.  Choisissez la pièce à exporter dans la [vue en arborescence](Tree_view/fr.md).
2.  Choisir **Raytracing** → **Utilitaires** → **<img src="images/Raytracing_WritePart.svg" width=32px> Exporter la pièce vers POV-Ray** dans le menu supérieur.
