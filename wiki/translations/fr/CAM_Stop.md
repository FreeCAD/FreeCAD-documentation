# CAM Stop/fr

## Description

L\'outil <img alt="" src=images/CAM_Stop.svg  style="width:24px;"> [Arrêter](CAM_Stop/fr.md) insère une commande d\'arrêt (M1). Cette commande suspendra le programme en cours d\'exécution sur le contrôleur CNC dans l\'attente d\'une nouvelle action de l\'utilisateur.

**Remarque : cela n\'arrête pas la broche.**



## Utilisation

1.  Pressez le bouton **<img src="images/CAM_Stop.svg" width=16px> [Arrêter](CAM_Stop/fr.md)
**



## Propriétés

-    **Stop**: définit si l\'arrêt est facultatif ou obligatoire.
