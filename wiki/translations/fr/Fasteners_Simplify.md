# Fasteners Simplify/fr

## Description

La commande <img alt="" src=images/Fasteners_Simplify.svg  style="width:24px;"> **Fasteners Simplifier la forme** crée des copies non-paramétriques des fixations. Cela peut être utile si vous souhaitez partager un document contenant des fixations avec des utilisateurs qui n\'ont pas installé l\'[atelier Fasteners](Fasteners_Workbench/fr.md).



## Utilisation

1.  Sélectionnez une ou plusieurs fixations.
2.  Il existe plusieurs façons de lancer la commande :
    -   Appuyez sur le bouton **<img src="images/Fasteners_Simplify.svg" width=16px> [Simplifier la forme](Fasteners_Simplify/fr.md)**.
    -   Sélectionnez l\'option **Fasteners → <img src="images/Fasteners_Simplify.svg" width=16px> Simplifier la forme** dans le menu.
3.  Pour chaque fixation, une copie simplifiée est créée.
4.  La propriété **Visibility** des fixations d\'origine est définie sur `False`.
5.  Si vous le souhaitez, vous pouvez supprimer les fixations d\'origine.
