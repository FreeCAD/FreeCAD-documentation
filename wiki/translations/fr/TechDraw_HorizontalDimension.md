# TechDraw HorizontalDimension/fr

## Description

L\'outil **TechDraw Cote horizontale** ajoute une cote horizontale à une vue. La cote peut être la distance horizontale entre deux points ou la longueur horizontale d\'une ligne droite. Comme pour l\'outil [TechDraw Lier une cote](TechDraw_LengthDimension/fr.md), il est également possible de sélectionner deux arêtes ou un point et une arête, mais ces options ont moins de sens pour cet outil.

Pour plus d\'informations, voir [TechDraw Lier une cote](TechDraw_LengthDimension/fr.md)
