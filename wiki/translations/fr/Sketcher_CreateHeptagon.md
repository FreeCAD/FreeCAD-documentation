# Sketcher CreateHeptagon/fr

## Description

L\'outil <img alt="" src=images/Sketcher_CreateHeptagon.svg  style="width:24px;"> [Sketcher Heptagone](Sketcher_CreateHeptagon/fr.md) crée un heptagone. {{Version/fr|1.0}} : il s\'agit du même outil que [Sketcher Polygone régulier](Sketcher_CreateRegularPolygon/fr.md) mais avec un nombre de côtés prédéfini à 7.

![](images/SketcherCreateHeptagonExample.png‎ )



## Utilisation

Voir aussi : [Aides au dessin](Sketcher_Workbench/fr#Aides_au_dessin.md).

1.  Il y a plusieurs façons de lancer l\'outil :
    -   Appuyez sur le bouton **<img src="images/Sketcher_CreateHeptagon.svg" width=16px> [Créer un heptagone](Sketcher_CreateHeptagon/fr.md)**.
    -   Sélectionnez l\'option **Esquisse → Géométries d'esquisse → <img src="images/Sketcher_CreateHeptagon.svg" width=16px> Créer un heptagone** du menu.
    -   Utilisez le raccourci clavier : **G** puis **P**, puis **7**.
2.  Pour d\'autres étapes, voir [Sketcher Polygone régulier](Sketcher_CreateRegularPolygon/fr#Utilisation.md).
