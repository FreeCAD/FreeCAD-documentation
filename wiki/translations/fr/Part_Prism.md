# Part Prism/fr

## Description

Un <img alt="" src=images/Part_Prism.svg  style="width:24px;"> **Part Prisme** est un solide paramétrique qui peut être créé avec la commande <img alt="" src=images/Part_Primitives.svg  style="width:24px;"> [Part Primitives](Part_Primitives/fr.md). Il est le résultat de l\'extrusion d\'un polygone régulier le long d\'une trajectoire droite. Dans le système de coordonnées défini par sa propriété **Placement**, la face inférieure du prisme se trouve sur le plan XY avec son centre à l\'origine et l\'un de ses sommets sur l\'axe X.

## Propriétés

Voir aussi : [Éditeur de propriétés](Property_editor/fr.md)

Un objet Part Prisme est dérivé d\'un [Part Feature](Part_Feature/fr.md) et hérite de toutes ses propriétés. Il possède également les propriétés supplémentaires suivantes :



### Données


{{TitleProperty|Attachment}}

L\'objet a les mêmes propriétés d\'attachement qu\'un [Part Part2DObject](Part_Part2DObject/fr#Donn.C3.A9es.md).


{{TitleProperty|Prism}}

-    **Polygon|IntegerConstraint**: nombre de côtés du polygone. La valeur par défaut est {{Value|6}}.

-    **Circumradius|Length**: rayon du cercle qui circonscrit le polygone, la distance entre le centre du polygone et l\'un de ses sommets. La valeur par défaut est {{Value|2mm}}.

-    **Height|Length**: hauteur du prisme. La valeur par défaut est {{Value|10mm}}.

-    **First Angle|Angle**: angle entre la direction d\'extrusion du prisme et son axe Z positif, mesuré autour de son axe Y. L\'angle est positif vers son axe X positif. L\'angle est positif vers son axe X positif. Plage valide : {{Value|0° &lt;&#61; value &lt; 90°}}. La valeur par défaut est {{Value|0°}}.

-    **Second Angle|Angle**: angle entre la direction d\'extrusion du prisme et son axe Z positif, mesuré autour de son axe X. L\'angle est positif vers son axe Y positif. L\'angle est positif vers son axe Y positif. Plage valide : {{Value|0° &lt;&#61; value &lt; 90°}}. La valeur par défaut est {{Value|0°}}.



## Script

Voir aussi : [Autogenerated API documentation](https://freecad.github.io/SourceDoc/), [Part Ecrire un script](Part_scripting/fr.md) et [Débuter avec les scripts FreeCAD](FreeCAD_Scripting_Basics/fr.md).

Un Part Prisme est créé avec la méthode `addObject()` du document :


```python
prism = FreeCAD.ActiveDocument.addObject("Part::Prism", "myPrism")
```

-   Où {{Incode|"myPrism"}} est le nom de l\'objet.
-   La fonction restitue l\'objet nouvellement créé.

Exemple :


```python
import FreeCAD as App

doc = App.activeDocument()

prism = doc.addObject("Part::Prism", "myPrism")
prism.Polygon = 5
prism.Circumradius = 10
prism.Height = 50
prism.FirstAngle = 22.5
prism.SecondAngle = 45
prism.Placement = App.Placement(App.Vector(1, 2, 3), App.Rotation(60, 75, 30))

doc.recompute()
```
