## Page des outils 

Pour chaque développement de logiciels sérieux, vous avez besoin d\'outils sérieux. Voici une liste d\'outils, que nous utilisons pour développer FreeCAD :

### Outils indépendants de la plate-forme 

#### Qt-Toolkit 

Qt-toolkit est un outil de conception d\'interface utilisateur à la pointe de la technologie et indépendant de la plate-forme. Elle est contenue dans le [LibPack](Third_Party_Libraries/fr.md) de FreeCAD, mais peut aussi être téléchargé à l\'adresse [Qt project](http://qt-project.org/downloads).

#### Inkscape

Il s\'agit d\'un excellent programme de dessin vectoriel. Il adhère à la norme SVG et est utilisé pour dessiner des icônes et des images. Inkscape est un logiciel libre et open source sous licence GPL. Vous pouvez l\'obtenir à l\'adresse [www.inkscape.org](http://www.inkscape.org).

#### Doxygen

Un très bon outil, stable, il génère de la documentation à partir de fichiers sources **.h** et **.cpp**.

#### Gimp

Il n\'y a pas grand chose à dire sur **GNU Image Manipulation Program**. Il peut manipuler les fichiers **.xpm**, ce qui est un moyen très pratique de gérer les icônes dans les programmes QT. XPM est en fait du code C qui peut être compilé dans un programme. Gimp supporte le format de fichier libre .png utilisé dans le wiki.

Téléchargez la dernière version [www.gimp.org](http://www.gimp.org/)

#### Gitkraken

C\'est une interface graphique propriétaire de git qui est très utile pour apprendre le fonctionnement de git si vous souhaitez contribuer à FreeCAD. Pages pertinentes à rechercher:

-   [Gitkraken](Developing_FreeCAD_with_GitKraken/fr.md)
-   [Github](Source_code_management/fr.md)

#### OBS Studio 

Logiciel libre et open source pour l\'enregistrement de vidéos et la diffusion en direct. Plus de détails sur le site officiel: [obsproject.com](https://obsproject.com/)

#### OpenCamLib

[OpenCamLib](OpenCamLib/fr.md) (OCL) est une bibliothèque open source visant à fournir des algorithmes de fabrication assistée par ordinateur (FAO). FreeCAD utilise OCL dans les opérations expérimentales de chemin de surface 3D et d\'autres fonctionnalités expérimentales.

#### StarUML

Un programme UML complet mais propriétaire. Il possède de nombreuses fonctionnalités majeures, dont la rétro-ingénierie du code source C++.

Téléchargez le ici : [staruml.io](http://staruml.io)

### Outils pour Windows 

#### Visual Studio Community 

Visual Studio Community est gratuit pour les projets open source et les petites structures. L\'installation est modulaire. Vous pouvez ajouter le support d\'un grand nombre de langages dont C++ et Python.

Vous pouvez le télécharger sur [Visual Studio Community](https://visualstudio.microsoft.com/fr/vs/community).

#### CamStudio

Outil Open Source pour enregistrer des Screen-casts (Webcasts). C\'est un très bon outil pour créer des tutoriels en les enregistrant. C\'est loin d\'être aussi ennuyeux que d\'écrire de la documentation.

Voir [camstudio.org](http://camstudio.org/) pour plus de détails.
