# Sketcher Create3PointCircle/fr

## Description

L\'outil <img alt="" src=images/Sketcher_Create3PointCircle.svg  style="width:24px;"> [Sketcher Cercle par 3 points](Sketcher_Create3PointCircle.md) crée un cercle à partir de trois points du cercle. {{Version/fr|1.0}} : il s\'agit du même outil que [Sketcher Cercle par centre](Sketcher_CreateCircle/fr.md) mais avec un mode initial différent.



## Utilisation

Voir aussi : [Aides au dessin](Sketcher_Workbench/fr#Aides_au_dessin.md).

1.  Il y a plusieurs façons de lancer l\'outil :
    -   Appuyez sur le bouton **<img src="images/Sketcher_Create3PointCircle.svg" width=16px> [Cercle par 3 points de la circonférence](Sketcher_Create3PointCircle/fr.md)**.
    -   Sélectionnez l\'option **Esquisse → Géometries d'esquisse → <img src="images/Sketcher_Create3PointCircle.svg" width=16px> Créer un cercle par 3 points** du menu.
    -   Utilisez le raccourci clavier : **G** puis **3**, puis **C**.
2.  Pour d\'autres étapes, voir [Sketcher Cercle par centre](Sketcher_CreateCircle/fr#Utilisation.md).
