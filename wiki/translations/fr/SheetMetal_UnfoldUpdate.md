# SheetMetal UnfoldUpdate/fr

## Description

La commande <img alt="" src=images/SheetMetal_UnfoldUpdate.svg  style="width:24px;"> [Mettre à jour les dépliés](SheetMetal_UnfoldUpdate/fr.md) met à jour tous les objets dépliés.



## Utilisation

1.  Il y a plusieurs façons de lancer la commande :
    -   Appuyez sur le bouton **<img src="images/SheetMetal_UnfoldUpdate.svg_" width=16px> [Mettre à jour les dépliés](SheetMetal_UnfoldUpdate/fr.md)**.
    -   Sélectionnez l\'option **SheetMetal → <img src="images/SheetMetal_UnfoldUpdate.svg" width=16px> Mettre à jour les dépliés** du menu.
    -   Cliquez avec le bouton droit de la souris dans la [vue arborescente](Tree_view/fr.md) ou dans la [vue 3D](3D_view/fr.md) et sélectionnez l\'option **SheetMetal → <img src="images/SheetMetal_UnfoldUpdate.svg" width=16px> Mettre à jour les dépliés** dans le menu contextuel.
2.  Tous les objets **dépliés** sont mis à jour.
