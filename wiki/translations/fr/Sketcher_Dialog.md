## Introduction

Lorsqu\'une esquisse est en cours d\'édition, la boîte de dialogue de Sketcher s\'affiche dans l\'onglet Tâche de la [vue combinée](Combo_view/fr.md).

Pour mettre une esquisse en mode édition, effectuez l\'une des opérations suivantes :

-   Double-cliquer sur l\'esquisse dans la [vue en arborescence](Tree_view/fr.md).
-   Cliquer avec le bouton droit de la souris sur l\'esquisse dans l\'arborescence et sélectionner **[Modifier l'esquisse](Sketcher_EditSketch/fr.md)** dans le menu contextuel.
-   Sélectionner l\'esquisse et appuyer sur **<img src="images/Sketcher_EditSketch.svg" width=16px> [Modifier l'esquisse](Sketcher_EditSketch/fr.md)**.
-   Sélectionner l\'esquisse, puis sélectionner **<img src="images/Sketcher_EditSketch.svg" width=16px>[Modifier l'esquisse](Sketcher_EditSketch/fr.md)** du menu.

La boîte de dialogue comprend les sections suivantes :



## Messages du solveur 

Cette section renseigne sur l\'état de l\'esquisse. Une esquisse peut être sous contrainte, entièrement contrainte ou avoir des contraintes redondantes.

Si une esquisse est sous contrainte, le nombre de degrés de liberté (ou Dofs) est indiqué. En cliquant sur ce texte, les éléments de l\'esquisse sous contrainte sont sélectionnés.

Si un esquisse a des contraintes redondantes, les indices de ces contraintes sont indiqués. Un clic sur ce texte sélectionnera ces contraintes.
