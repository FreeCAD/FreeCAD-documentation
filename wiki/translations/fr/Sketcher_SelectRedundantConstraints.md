# Sketcher SelectRedundantConstraints/fr

## Description

Sélectionne les contraintes redondantes d\'une esquisse.



## Utilisation

1.  Entrez dans le mode d\'édition de l\'esquisse.
2.  Choisissez **Esquisse → Outils d'esquisse → [<img src=images/Sketcher_SelectRedundantConstraints.svg style="width:16px"> Sélectionner les contraintes redondantes** dans le menu principal.
3.  Les contraintes redondantes de l\'esquisse seront sélectionnées.
