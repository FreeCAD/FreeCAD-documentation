# Spreadsheet CreateSheet/fr

## Description

L\'outil **[<img src=images/Spreadsheet_CreateSheet.svg style="width:16px"> [Spreadsheet Créer une feuille de calcul](Spreadsheet_CreateSheet/fr.md)** ajoute un nouvel objet de feuille de calcul au document actif qui sera ouvert dans un nouvel onglet dans la [Zone de vue principale](main_view_area/fr.md).

## Utilisation

1.  Appuyez sur le bouton **[<img src=images/Spreadsheet_CreateSheet.svg style="width:16px"> [Créer une feuille de calcul](Spreadsheet_CreateSheet.md)**.
2.  Modifiez le contenu des cellules.
3.  Fermez l\'onglet de la feuille de calcul lorsque vous avez terminé, les valeurs sont enregistrées automatiquement lorsque vous les tapez.
