# TechDraw ExtensionCustomizeFormat/fr

## Description

L\'outil <img alt="" src=images/TechDraw_ExtensionCustomizeFormat.svg  style="width   *24px;"> **TechDraw Personnaliser le format d\'infobulle** personnalise le formatage du texte d\'une bulle ou d\'une cote. Il est possible d\'ajouter des [symboles GD&T](TechDraw_Geometric_dimensioning_and_tolerancing/fr.md) et d\'autres caractères spéciaux.

<img alt="" src=images/TechDraw_ExtensionCustomizeFormatExample.png  style="width   *250px;"> 
*Le panneau de tâches Personnaliser le format*

## Utilisation

1.  Sélectionnez un objet Infobulle ou un objet Dimension.
2.  Il existe plusieurs façons de lancer l\'outil    *
    -   Appuyez sur le bouton **<img src="images/TechDraw_ExtensionCustomizeFormat.svg" width=16px> [Personnaliser le format de l'infobulle](TechDraw_ExtensionCustomizeFormat/fr.md)** bouton.
    -   Sélectionnez l\'option **TechDraw → Extensions    * attributs/modifications → <img src="images/TechDraw_ExtensionCustomizeFormat.svg" width=16px> Personnaliser le format de l'infobulle** dans le menu.
3.  Le panneau de tâches **Personnaliser le format** s\'ouvre.
4.  Modifiez le champ **Format** à l\'aide des boutons disponibles du panneau de tâches et/ou du clavier.
5.  Le champ **Aperçu** affiche le résultat du formatage en cours.
6.  Appuyez sur le bouton **OK**.
