# TechDraw StackBottom/fr

## Description

L\'outil <img alt="" src=images/TechDraw_StackTop.svg  style="width   *24px;"> **TechDraw Empiler en bas** déplace les vues vers le bas de l\'ordre d\'empilement. L\'ordre d\'empilement contrôle la profondeur apparente des vues sur une page.
