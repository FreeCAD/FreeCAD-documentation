# Std ViewZoomIn/fr

## Description

La commande **Std Zoom avant** effectue un zoom avant dans la [vue 3D](3D_view/fr.md) active.



## Utilisation

1.  Il existe plusieurs façons de lancer la commande :
    -   Sélectionnez l\'option **Affichage → Zoom → <img src="images/Std_ViewZoomIn.svg" width=16px> Zoom avant** du menu.
    -   Utilisez le raccourci clavier: **Ctrl**+**+**.



## Remarques

-   Avec presque tous les [styles de navigation à la souris](Mouse_navigation/fr.md), il est également possible de zoomer à l\'aide de la molette de la souris.



## Préférences

Voir aussi : [Réglage des préférences](Preferences_Editor/fr.md).

-   Le facteur de zoom peut être modifié : **Édition → Préférences... → Affichage → Navigation → Pas du zoom**. Ce paramètre affecte également le zoom de la molette de défilement.



## Script

Voir aussi : [Autogenerated API documentation](https://freecad.github.io/SourceDoc/) et [FreeCAD Débuter avec les scripts](FreeCAD_Scripting_Basics/fr.md).

Utilisez la méthode `zoomIn` de l\'objet View pour effectuer un zoom avant. La méthode `zoomOut` est également disponible.


```python
import FreeCADGui

view = FreeCADGui.ActiveDocument.ActiveView
view.zoomIn()
```
