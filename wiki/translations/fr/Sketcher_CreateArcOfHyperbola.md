# Sketcher CreateArcOfHyperbola/fr

## Description

Cet outil créé un arc d\'hyperbole par son centre, son rayon majeur et les points d\'extrémités.

![](images/Sketcher_Arc_of_Hyperbola_01.png )

## Utilisation

1.  Appuyer sur le bouton **<img src=images/Sketcher_CreateArcOfHyperbola.svg style="width:16px"> [Créer un arc d'hyperbole](Sketcher_CreateArcOfHyperbola/fr.md)**.
2.  Cliquez dans la vue 3D pour définir le centre du rayon principal (numéroté 1 dans l\'image ci-dessus).
3.  Cliquez dans la vue 3D pour définir la longueur du rayon principal, qui correspond également à la crête de l\'arc d\'hyperbole (numéroté 2 dans l\'image ci-dessus).
4.  Éloignez la souris des deux derniers points sélectionnés pour voir à quoi ressemblera l\'arc. Cliquez à nouveau lorsque la courbure approximative souhaitée est affichée. Ce point définira le premier sommet (numéroté 3 dans l\'image ci-dessus).
5.  Cliquez une quatrième fois dans la vue 3D pour définir l\'autre sommet (numéroté 4 dans l\'image ci-dessus).
