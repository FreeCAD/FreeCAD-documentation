# CAM Comment/fr

## Description

L\'outil <img alt="" src=images/CAM_Comment.svg  style="width:24px;"> [Commentaire](CAM_Comment/fr.md) insère un commentaire. Lors de l\'export d\'un projet en G-code, le commentaire sera inclus dans le programme G-code et peut être lu par les personnes qui regardent le code. Certains contrôleurs de machine les affichent aussi sur leurs consoles.



## Utilisation

1.  Pressez le bouton **<img src="images/CAM_Comment.svg" width=16px> [Commenter](CAM_Comment/fr.md)** ou utilisez le raccourci clavier **P** puis **C**.
2.  Écrivez votre commentaire dans la zone **Commentaire**



## Propriétés

-    **Comment**: le texte de votre commentaire
