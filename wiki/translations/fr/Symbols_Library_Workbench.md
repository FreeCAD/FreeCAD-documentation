# Symbols Library Workbench/fr
## Introduction


{{TOCright}}

Ce dépôt contient une bibliothèque de symboles SVG à utiliser dans FreeCAD. Bien qu\'ils soient constitués de fichiers SVG simples, ils peuvent donc également être importés dans le document 3D, ils sont principalement destinés à être utilisés sur les pages de dessin.

Il est maintenu par la communauté des utilisateurs de FreeCAD et ne fait pas partie du projet FreeCAD, bien qu\'il soit conçu dans le but de devenir un dépot de pièces par FreeCAD à l\'avenir.

## Installation



### Installation Automatique 

Cet atelier peut être installé à partir du [Gestionnaire des extensions](Std_AddonMgr/fr.md).



### Depuis GitHub 

La bibliothèque est un simple conteneur pour les fichiers SVG. Vous pouvez le télécharger n\'importe où et importer ses fichiers dans vos projets FreeCAD. Dans la bibliothèque, il y a aussi une macro FreeCAD (SymbolsLibrary.FCMacro) que vous pouvez modifier (vous devez changer le chemin de votre bibliothèque) et placer dans votre dossier de macros FreeCAD. Cette macro crée une fenêtre de navigateur dans FreeCAD, à partir de laquelle vous pouvez facilement ajouter les symboles à une page de dessin sélectionnée (ou, si aucun n\'est sélectionné, la première trouvée dans le document actif) en double-cliquant dessus.
