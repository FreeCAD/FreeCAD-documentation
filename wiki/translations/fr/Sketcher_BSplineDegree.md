# Sketcher BSplineDegree/fr

## Description

L\'outil <img alt="" src=images/Sketcher_BSplineDegree.svg  style="width:24px;"> [Sketcher Degré d\'une B-spline](Sketcher_BSplineDegree/fr.md) permet d\'afficher ou de masquer le degré des [B-splines](B-Splines.md) dans toutes les esquisses.

![](images/Sketcher_B-spline_example01.png ) 
*Le chiffre 3 (vert, sans parenthèses) au centre correspond au [degré](Sketcher_BSplineIncreaseDegree/fr.md) de la B-spline.*



## Utilisation

1.  Il y a plusieurs façons de lancer l\'outil :
    -   Appuyez sur le bouton **<img src="images/Sketcher_BSplineDegree.svg" width=16px> [Afficher/masquer le degré d'une B-spline](Sketcher_BSplineDegree/fr.md)**.
    -   Sélectionnez l\'option **Esquisse → Affichage → Afficher/masquer les informations d'une B-spline → <img src="images/Sketcher_BSplineDegree.svg" width=16px> Afficher/masquer le degré d'une B-spline** du menu.
