# TechDraw StackDown/fr

## Description

L\'outil <img alt="" src=images/TechDraw_StackDown.svg  style="width   *24px;"> **TechDraw Empiler vers le bas** déplace les vues d\'un niveau inférieur dans l\'ordre d\'empilement. L\'ordre d\'empilement contrôle la profondeur apparente des vues sur une page.

Voir [TechDraw Empiler en haut](TechDraw_StackTop/fr.md).
