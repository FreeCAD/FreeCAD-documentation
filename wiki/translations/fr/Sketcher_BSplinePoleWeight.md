# Sketcher BSplinePoleWeight/fr

## Description

L\'outil <img alt="" src=images/Sketcher_BSplinePoleWeight.svg  style="width:24px;"> [Sketcher Poids des points de contrôle B-spline](Sketcher_BSplinePoleWeight/fr.md) affiche ou masque le poids des points de contrôle des [B-splines](B-Splines/fr.md) dans toutes les esquisses.

<img alt="" src=images/sketcher_BSplineWeightShow.png  style="width:468px;"> 
*Les chiffres (verts, entre crochets) correspondent aux poids des points de contrôle.*



## Utilisation

1.  Il y a plusieurs façons de lancer l\'outil :
    -   Appuyez sur le bouton **<img src="images/Sketcher_BSplinePoleWeight.svg" width=16px> [Afficher/masquer le poids du point de contrôle d'une B-spline](Sketcher_BSplinePoleWeight/fr.md)**.
    -   Sélectionnez l\'option **Esquisse → Affichage → Afficher/masquer les informations d'une B-spline → <img src="images/Sketcher_BSplinePoleWeight.svg" width=16px> Afficher/masquer le poids du point de contrôle d'une B-spline** du menu.



## Changer des poids 

La façon de modifier les poids est décrite sur [cette page](B-Splines#Changing_the_Weight.md).
