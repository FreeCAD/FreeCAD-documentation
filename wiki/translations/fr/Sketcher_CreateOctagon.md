# Sketcher CreateOctagon/fr

## Description

L\'outil <img alt="" src=images/Sketcher_CreateOctagon.svg  style="width:24px;"> [Sketcher Octogone](Sketcher_CreateOctagon/fr.md) crée un octogone. {{Version/fr|1.0}} : il s\'agit du même outil que [Sketcher Polygone régulier](Sketcher_CreateRegularPolygon/fr.md) mais avec un nombre de côtés prédéfini à 8.

![](images/SketcherCreateOctagonExample.png‎ )



## Utilisation

Voir aussi : [Aides au dessin](Sketcher_Workbench/fr#Aides_au_dessin.md).

1.  Il y a plusieurs façons de lancer l\'outil :
    -   Appuyez sur le bouton **<img src="images/Sketcher_CreateOctagon.svg" width=16px> [Créer un heptagone](Sketcher_CreateOctagon/fr.md)**.
    -   Sélectionnez l\'option **Esquisse → Géométries d'esquisse → <img src="images/Sketcher_CreateOctagon.svg" width=16px> Créer un octogone** du menu.
    -   Utilisez le raccourci clavier : **G** puis **P**, puis **8**.
2.  Pour d\'autres étapes, voir [Sketcher Polygone régulier](Sketcher_CreateRegularPolygon/fr#Utilisation.md).
