# BIM SetWPFront/fr



## Description

L\'outil <img alt="" src=images/BIM_SetWPFront.svg  style="width:24px;"> [BIM Plan de travail de devant](BIM_SetWPFront/fr.md) place le plan de travail sur le plan XZ global (devant).



## Utilisation

1.  Il y a plusieurs façons de lancer l\'outil :
    -   Sélectionnez l\'option **Aimantation → <img src="images/BIM_SetWPFront.svg" width=16px> Plan de travail de devant** du menu.
    -   Cliquez avec le bouton droit de la souris dans la [vue 3D](3D_view/fr.md) et sélectionnez l\'option **Aimantation → <img src="images/BIM_SetWPFront.svg" width=16px> Plan de travail de devant** du menu contextuel.
    -   Utilisez le raccourci clavier : **W** puis **P**, puis **1**.
