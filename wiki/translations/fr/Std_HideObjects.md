# Std HideObjects/fr

## Description

La commande **Std Masquer tous les objets** masque tous les objets appartenant au document actif dans la [vue 3D](3D_view/fr.md).



## Utilisation

1.  Sélectionnez l\'option **Affichage → Visibilité → <img src="images/Std_HideObjects.svg" width=16px> Masquer tous les objets** du menu.
