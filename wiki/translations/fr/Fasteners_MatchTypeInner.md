# Fasteners MatchTypeInner/fr

## Description

La commande <img alt="" src=images/Fasteners_MatchTypeInner.svg  style="width:24px;"> **Fasteners Adapter pour le trou taraudé** se comporte comme une bascule avec la commande [Fasteners Adapter pour le trou de passage](Fasteners_MatchTypeOuter/fr.md). Lorsque la commande **Fasteners Adapter pour le trou taraudé** est sélectionnée, les bords circulaires sont considérés comme des trous taraudés lorsque de nouvelles fixations y sont fixées. Cela détermine les dimensions par défaut de ces fixations. Ce paramètre n\'est pas utilisé pour les vis à tête fraisée.



## Utilisation

1.  Il existe plusieurs façons de lancer la commande :
    -   Appuyez sur le bouton **<img src="images/Fasteners_MatchTypeInner.svg" width=16px> [Adapter pour le trou taraudé](Fasteners_MatchTypeInner/fr.md)**.
    -   Sélectionnez l\'option **Fasteners → <img src="images/Fasteners_MatchTypeInner.svg" width=16px> Adapter pour le trou taraudé** du menu.
2.  Le bouton reste enfoncé pour indiquer l\'option sélectionnée.
