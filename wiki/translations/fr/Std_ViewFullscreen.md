# Std ViewFullscreen/fr

## Description

La commande **Std Plein écran** bascule la [vue 3D](3D_view/fr.md) vers et depuis le mode plein écran. Les vues 3D sont automatiquement [désancrées](Std_ViewDockUndockFullscreen/fr.md) dans le mode plein écran.



## Utilisation

1.  Activez la bonne vue 3D.
2.  Il existe plusieurs façons de lancer la commande :
    -   Sélectionnez l\'option **Affichage → Fenêtre de document → <img src="images/Std_ViewFullscreen.svg" width=16px> Plein écran** du menu. Cette option ne fonctionne pas si vous souhaitez désactiver le mode plein écran et qu\'il existe d\'autres vues 3D ancrées.
    -   Sélectionnez l\'option **Fenêtre de document → <img src="images/Std_ViewFullscreen.svg" width=16px> Plein écran** dans le menu contextuel de la vue 3D.
    -   Utilisez le raccourci clavier : **F11**.



## Remarques

-   Plusieurs vues 3D pour le même document peuvent être créées avec la commande [ Std Créer une nouvelle vue](Std_ViewCreate/fr.md).
-   Les vues 3D peuvent également être ancrées et désancrées avec la commande [Std Désancrer une fenêtre de document](Std_ViewDockUndockFullscreen/fr.md).
