# Spreadsheet SplitCell/fr

## Description

L\'outil <img alt="" src=images/Spreadsheet_SplitCell.svg  style="width:24px;"> **Spreadsheet Diviser une cellule** divise une cellule préalablement fusionnée.



## Utilisation

1.  Assurez-vous qu\'il existe une [feuille de calcul](Spreadsheet_CreateSheet/fr.md) active.
2.  Sélectionnez une cellule [fusionnée](Spreadsheet_MergeCells/fr.md).
3.  Il y a plusieurs façons de lancer l\'outil :
    -   Appuyez sur le bouton **[<img src=images/Spreadsheet_SplitCell.svg style="width:16px"> [Diviser la cellule](Spreadsheet_SplitCell/fr.md)**.
    -   Sélectionnez l\'option **Spreadsheet → [<img src=images/Spreadsheet_SplitCell.svg style="width:16px"> Diviser la cellule** du menu.
