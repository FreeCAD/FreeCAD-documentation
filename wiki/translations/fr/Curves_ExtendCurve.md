## Description

<img alt="" src=images/Curves_ExtendCurve.svg  style="width:24px;"> [Extension de courbe](Curves_ExtendCurve/fr.md) étend le bord sélectionné. Cet outil fait partie des [ateliers externes](External_workbenches/fr.md) appelé [Curves](Curves_Workbench/fr.md).

## Utilisation

1.  Passer à l\'atelier <img alt="" src=images/Curves_workbench_icon.svg  style="width:24px;"> [Curves](Curves_Workbench/fr.md) (à installer à partir du <img alt="" src=images/Std_AddonMgr.svg  style="width:24px;"> [Gestionnaire d\'Addon](Std_AddonMgr/fr.md) si ce n\'est pas déjà fait)
2.  Pour appeler la commande, effectuez l\'une des opérations suivantes:
    -   Appuyez sur le bouton <img alt="" src=images/Curves_ExtendCurve.svg  style="width:24px;"> dans la barre d\'outils Curves.
    -   Utilisez l\'entrée **Curves → Extend Curve** dans le menu déroulant.
