# Robot RestoreHomePos/tr


</div>

## Description


<div class="mw-translate-fuzzy">

#### Açıklama

Seçilen robotun ana konumuna dönmesini sağlar.


</div>

## Usage


<div class="mw-translate-fuzzy">

#### Kullanımı

1.  Robotu ağaç görünümünde veya 3D görünümünde seçin
2.  ![ 32px](images/_Robot_RestoreHomePos.png ) üzerine tıklayın veya üst menüden {{KEY | Robot}} → {{KEY | <img src="images/_Robot_RestoreHomePos.png_" width= 32px>  Ana konuma dön}}\'ü seçin.


</div>

## Notes


<div class="mw-translate-fuzzy">

#### Notlar

Komut yalnızca bir robotun ana konumunu belirlemesine izin verir.
Birden fazla robot seçilmediyse, sadece bir robot seçmeniz istenir.
[Robot_Ana konum ayarla](Robot_SetHomePos/tr.md) üzerinden ana konum belirtilmemişse, robotun sahneye yerleştirildiği şekliyle ana konum kullanılır.


</div>
