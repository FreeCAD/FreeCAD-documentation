# Spreadsheet Preferences/pl
Preferencje dla środowiska pracy <img alt="" src=images/Workbench_Spreadsheet.svg  style="width:24px;"> [Arkusz Kalkulacyjny](Spreadsheet_Workbench/pl.md) *({{Version/pl|0.20}})* można znaleźć w [Edytorze preferencji](Preferences_Editor/pl.md). W menu wybierz **Edycja → Preferencje ...**, a następnie **<img src="images/Workbench_Spreadsheet.svg" width=16px> Arkusz Kalkulacyjny**. Ta grupa jest dostępna tylko jeśli środowisko pracy Arkusz Kalkulacyjny zostało załadowane w bieżącej sesji programu FreeCAD.

Istnieje tylko jedna strona: *Arkusz kalkulacyjny*.

![](images/Preferences_Spreadsheet_Page_Spreadsheet.png )

Na tej stronie można określić następujące parametry:

+++
| Nazwa                                               | Opis                                                                                                                                                                                                                     |
+=====================================================+==========================================================================================================================================================================================================================+
|                                      | Znak używany jako ogranicznik pola.                                                                                                                                                                                      |
| **Znak separatora**                     |                                                                                                                                                                                                                          |
|                                                  |                                                                                                                                                                                                                          |
+++
|                                      | Znak używany do określania ciągów znaków.                                                                                                                                                                                |
| **Znak cytatu**                         |                                                                                                                                                                                                                          |
|                                                  |                                                                                                                                                                                                                          |
+++
|                                      | Znak używany do zapobiegania przetwarzaniu następującego znaku.                                                                                                                                                          |
| **Znak Escape**                         |                                                                                                                                                                                                                          |
|                                                  |                                                                                                                                                                                                                          |
+++
|                                      | Jeśli zaznaczone i jeśli komórka ma wartość i alias to są one wyświetlane w podanym formacie. {{Value|%V}} określa wartość, zaś {{Value|%A}} określa alias. {{Version/pl|1.0}} |
| **Pokaż alias w komórce sformatowanej** |                                                                                                                                                                                                                          |
|                                                  |                                                                                                                                                                                                                          |
+++
