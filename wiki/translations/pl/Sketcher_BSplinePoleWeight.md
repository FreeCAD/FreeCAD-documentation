# Sketcher BSplinePoleWeight/pl



## Opis

Narzędzie <img alt="" src=images/Sketcher_BSplinePoleWeight.svg  style="width:24px;"> **Pokaż / ukryj wagę punktu kontrolnego krzywej złożonej** pokazuje lub ukrywa wagę punktu kontrolnego [krzywej złożonej](B-Splines/pl.md) we wszystkich szkicach.

<img alt="" src=images/sketcher_BSplineWeightShow.png  style="width:468px;"> 
*Liczby ''(zielone, w nawiasach kwadratowych)'' odnoszą się do wag punktów kontrolnych.*



## Użycie

1.  Istnieje kilka sposobów wywołania narzędzia:
    -   Naciśnij przycisk **<img src="images/Sketcher_BSplinePoleWeight.svg" width=16px> '''Pokaż / ukryj wagę punktu kontrolnego krzywej złożonej'''**.
    -   Wybierz z menu **Szkic → Widok szkicu → Pokaż / ukryj warstwę informacyjną krzywej złożonej → <img src="images/Sketcher_BSplinePoleWeight.svg" width=16px> Pokaż / ukryj wagę punktu kontrolnego krzywej złożonej**.



## Zmiana wag 

Sposób zmiany wag jest opisany na stronie przedstawiającej [Krzywe złożone](B-Splines/pl#Zmiana_wagi.md).
