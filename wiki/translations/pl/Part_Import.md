# Part Import/pl



## Opis

Narzędzie **<img src="images/Part_Import.svg" width=16px> '''Import projektu CAD ...'''** otwiera okno dialogowe umożliwiające import następujących formatów plików CAD: [IGES](http://en.wikipedia.org/wiki/IGES), [STEP](http://en.wikipedia.org/wiki/Step_file) i [BREP](http://en.wikipedia.org/wiki/BREP) do bieżącego dokumentu.



## Użycie

1.  Skonfiguruj [Ustawienia Importu i Eksportu](Import_Export_Preferences/pl.md) w [Edytor ustawień](Preferences_Editor/pl.md).
2.  Przejdź do menu **Część → [<img src=images/Part_Import.svg style="width:16px"> Import projektu CAD**.
3.  Wybierz plik do zaimportowania.
