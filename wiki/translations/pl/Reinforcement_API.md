# Reinforcement API/pl
**''(listopad 2018)'' Dodatek [Zbrojenie](Reinforcement_Workbench/pl.md) jest rozwijany oddzielnie od [Środowiska pracy Architektura](Arch_Workbench/pl.md). Jego kod i API są hostowane w repozytorium [https://github.com/amrit3701/FreeCAD-Reinforcement FreeCAD-Reinforcement], które jest oddzielone od głównego repozytorium programu FreeCAD.**

Zobacz także stronę [Architektura: API](Arch_API/pl.md) dla głównych funkcji środowiska pracy [Architektura](Arch_Workbench/pl.md).

Funkcje API mogą być używane w [makrodefinicjach](Macros/pl.md) i z konsoli [Python](Python/pl.md) po zaimportowaniu odpowiedniego modułu.


```python
import StraightRebar
import UShapeRebar
import LShapeRebar
import BentShapeRebar
import Stirrup
import HelicalRebar
```
