# Path Stop/pl

## Description

The tool <img alt="" src=images/Path_Stop.svg  style="width:24px;"> [Stop](Path_Stop.md) inserts a Stop command (M1). This command will Pause a running program on the CNC controller, waiting for user interaction to continue.

**Note: this does not stop the spindle.**

## Usage

1.  Press the **<img src="images/Path_Stop.svg" width=16px> [Stop](Path_Stop.md)** button

## Properties

-    **Stop**: Defines if the Stop is optional or mandatory.
