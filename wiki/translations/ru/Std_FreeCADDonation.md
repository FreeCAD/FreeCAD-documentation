# Std FreeCADDonation/ru



## Описание


<div lang="en" dir="ltr" class="mw-content-ltr">

The **Std FreeCADDonation** command opens the FreeCAD Donate wiki page in the system\'s default Internet browser. The command requires a working Internet connection.


</div>

На текущий момент команда направляет пользователя на wiki страницу: [<https://wiki.freecad.org/Donate>](https://wiki.freecad.org/Donate).



## Применение

1.  Выберите пункт меню **Помощь → <img src="images/Std_FreeCADDonation.svg" width=16px> Поддержать**.
