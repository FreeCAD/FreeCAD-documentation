# Std Cut/ru

## Описание

Команда \"Вырезать\" имеет ограничения   * ее можно использовать только при работе с [ячейками электронных таблиц](Spreadsheet_Workbench/ru.md). Команда копирует содержимое и свойства ячеек в буфер обмена, а затем очищает ячейки.

Чтобы вырезать объекты других типов, вы можете использовать команду [Копировать](Std_Copy/ru.md), следом за которой выполните удаление объекта.

## Применение

1.  Выберите одну или несколько ячеек электронной таблицы.
2.  Существует несколько способов для вызова данной команды   *
    -   Нажатием кнопки **<img src="images/Std_Cut.svg" width=16px> [Вырезать](Std_Cut/ru.md)** на панели инструментов.
    -   Через пункт меню **Правка → <img src="images/Std_Cut.svg" width=16px> Вырезать
**

## Примечания

-   Стандартное сочетание клавиш **Ctrl**+**X** не вызывает команду \"Std\_Cut\", а вместо этого использует аналогичную по своей сути функцию, но из операционной системы. Вы можете использовать это сочетание клавиш в большинстве текстовых окон и полей ввода FreeCAD, в том числе и для правки ячеек электронной таблицы.
