## Описание

The <img alt="" src=images/Draft_PathLinkArray.svg  style="width:24px;"> **Draft PathLinkArray** command creates a [Link](App_Link.md) array from a selected object by placing copies along a path. Use the [Draft PathArray](Draft_PathArray.md) command to create a less efficient regular array instead. Except for the type of array that is created, [Link](App_Link.md) array or regular array, this command is identical to the [Draft PathArray](Draft_PathArray.md) command. See there for more information.
