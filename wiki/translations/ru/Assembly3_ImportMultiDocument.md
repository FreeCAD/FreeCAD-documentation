# Assembly3 ImportMultiDocument/ru



## Описание

The <img alt="" src=images/Assembly_ImportMulti.svg‎‎  style="width:24px;"> [Import as multi-document](Assembly3_ImportMultiDocument.md) command opens a file browser window to search and select STEP files.

STEP assemblies are imported into separate documents.



## Применение

1.  There are several ways to invoke the command:
    -   Depending on the button condition do one of the following:
        -   Press the icon of the **<img src="images/Assembly_ImportMulti.svg‎‎" width=16px><img src="images/Toolbar_flyout_arrow.svg" width=x16px> [Import as multi-document](Assembly3_ImportMultiDocument.md)** button.
        -   Press the down arrow of the **<img src="images/Assembly_Import.svg" width=|x16px><img src="images/Toolbar_flyout_arrow.svg" width=x16px> '''Import from STEP'''** button and select the **<img src="images/Assembly_ImportMulti.svg" width=16px> Import as multi-document** option from the dropdown.
    -   Select the **Assembly3 → Import from STEP → <img src="images/Assembly_ImportMulti.svg‎‎" width=16px> Import as multi-document** option from the menu.
    -   Select the **Assembly → Import from STEP → <img src="images/Assembly_ImportMulti.svg‎‎" width=16px> Import as multi-document** option from the [Tree view](Tree_view.md) context menu or the [3D view](3D_view.md) context menu.
2.  A file browser opens.
3.  Select the STEP file to import.
