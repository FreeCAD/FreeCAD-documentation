# Raytracing Render/ru


</div>

## Description

Renders the selected raytracing project using an external renderer.

## Usage

1.  Select raytracing project in the [tree view](Tree_view.md).
2.  Click on <img alt="" src=images/Raytrace_Render.svg  style="width:32px;"> or choose ** Raytracing** → **<img src="images/Raytrace_Render.svg" width=32px> Render** from the top menu.

## Limitations

Currently only POV-Ray and Luxrender are supported.
