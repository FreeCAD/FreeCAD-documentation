# OpenSCAD IncreaseToleranceFeature/ru


</div>

## Description

Увеличивает допуски для ребер/граней/вершины выбранного объекта (ов).

## Usage

1.  Выбрать объект в [древе проекта](Tree_view/ru.md)
2.  Выбрать в верхнем меню **OpenSCAD** → **<img src="images/OpenSCAD_IncreaseToleranceFeature.svg" width=24px> Increase Tolerance Feature**.
3.  Выбрать Tolerance-Object в древе свойств в [древе проекта](Tree_view/ru.md).
4.  Выбрать вкладку Данные [Редакторе параметров](Property_editor/ru.md) or [окне комбинированного просмотра](Combo_view/ru.md).
5.  Установить желательные значения для рёбер/граней/вершин.
