# Std DlgPreferences/ru



## Описание

Команда **Настройки\...** открывает [Редактор настроек](Preferences_Editor/ru.md).



## Применение

1.  Выберите пункт меню **Правка → <img src="images/Std_DlgPreferences.svg" width=16px> Настройки...**.



## Примечания

-   For more information see [Preferences Editor](Preferences_Editor.md).
-   On macOS this command may appear in the FreeCAD menu instead of the [Edit](Std_Edit_Menu.md) menu.
