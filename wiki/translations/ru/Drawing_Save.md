# Drawing Save/ru

Этот инструмент сохраняет текущий чертёжный лист в файле SVG (scalable vector graphics). Это файл может редактироваться в таких редакторах векторной графики, как [Inkscape](http   *//www.inkscape.org).


<div class="mw-translate-fuzzy">

Файлы SVG стандартизированы и могут просматриваться в большинстве современных браузеров и просмотрщиков изображений. Это один из способов поделиться проектом с людьми, у которых FreeCAD не установлен на их ПЭВМ.
