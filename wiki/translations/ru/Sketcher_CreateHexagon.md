# Sketcher CreateHexagon/ru



## Описание

Создает шестиугольник, вписанный в окружность вспомогательной геометрии. При запуске инструмента указатель мыши меняется на белый крест с красным значком шестиугольника. Координаты указателя отображаются рядом с ним синим цветом в режиме реального времени.

![](images/SketcherCreateHexagonExample.png‎ )



## Применение

-   После нажатия кнопки **[<img src=images/Sketcher_CreateHexagon.svg style="width:16px"> [Шестиугольник](Sketcher_CreateHexagon.md)**, нажмите первый раз, чтобы задать центр, затем переместите мышь и нажмите второй раз, чтобы задать одну из вершин.
-   Нажатие **Esc** или правой кнопки мыши закрывает функцию.



## Примечания

Видимая окружность является элементом вспомогательной геометрией, которая видна только при редактировании эскиза.
