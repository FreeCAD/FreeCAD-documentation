# Drawing Symbol/ru

## Описание

Эта команда позволяет добавить содержимое файла SVG выбранный [чертёжный лист](Drawing_Landscape_A3.md). Это содержимое может перемещаться и масштабироваться на странице. Содержимое изображения SVG копируется внутрь документа FreeCAD, становясь независимым от оригинального файла, и изображается так же на других компьютерах, где нет оригинального файла SVG.

## Использование


</div>

1.  Create a [Drawing page](Drawing_Landscape_A3.md)
2.  [Refresh](Std_Refresh.md) the view if a Drawing viewer wasn\'t opened
3.  Press the **<img src="images/Drawing_Symbol.png" width=16px> [[Drawing Symbol]]** button
4.  Select a SVG file
5.  Adjust the needed properties, like position and scale.
