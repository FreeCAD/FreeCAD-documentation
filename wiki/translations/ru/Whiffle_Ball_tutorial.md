## Введение

Этот учебник был первоначально написан Роландом Франком , а затем переписан и проиллюстрирован vox.

Этот учебник создан чтобы научить вас использовать Верстак Part.

Верстак Part был первым разработанным верстаком. Он предоставляет основные геометрические элементы, которые могут быть использованы в качестве строительных блоков для других верстаков. Верстак Part предназначен для использования в традиционном рабочем процессе создания конструктивной твердотельной геометрии. Для более современного рабочего процесса с использованием эскизов, функции выдавливания и др. функций используйте верстак PartDesign.

Вы попрактикуетесь в:

-   вставке примитивов
-   изменении параметров этих примитивных объектов
-   изменении их размещения 
-   выполнении Булевых операций

! 
*Окончательная модель мяча для игры в Whiffle Ball*



## Настройка

1\. Откройте FreeCAD и создайте новый пустой документ с помощью **Файл, File:Std_New.svg   16px Std_New/ru**{: mediawiki}, и переключитесь на верстак Part.

:   1.1. Нажмите на кнопку**File:Std_ViewIsometric.svg   16px Std_ViewIsometric/ru**{: mediawiki}, или нажмите **0** на цифровой панели клавиатуры, для переключения в изометрический вид и лучшего представления 3D объектов.
:   1.2. Нажмите кнопку **File:Std_ViewFitAll.svg   16px Std_ViewFitAll/ru**{: mediawiki} всякий раз, когда вы добавляете объекты для панорамирования и масштабирования 3D-вида, чтобы все элементы были видны на экране.
:   1.3. Удерживайте нажатой клавишу **Ctrl**, для выбора нескольких элементов. Если вы выбрали что-то неправильно или хотите отменить выбор всего, просто нажмите на пустое место в 3D-виде.



## Вставка примитивных кубов 

2\. Вставьте примитивный куб, нажав на кнопку **Image:Part_Box.svg   16px Part_Box/ru**{: mediawiki}.

:   2.1. Выберите {{incode   Куб}}{: mediawiki} в древе проекта.
:   2.2. Измените размеры во вкладке **Данные** на панели Окна свойств.
:   2.3. Измените **Length** на{{incode   90 мм}}{: mediawiki}.
:   2.4. Измените **Width** на {{incode   90 мм}}{: mediawiki}.
:   2.5. Измените **Height** на {{incode   90 мм}}{: mediawiki}.

3\. На вкладке **Данные** панели Окна свойств, нажмите на **Placement** так что бы появилась кнопка **...** справа.

:   3.1. Нажмите на неё для запуска диалогового окна Placement.
:   3.2. Измените значения **Перемещения**.
:   3.3. Измените **X** на {{incode   -45 мм}}{: mediawiki}.
:   3.4. Измените **Y** на {{incode   -45 мм}}{: mediawiki}.
:   3.5. Измените **Z** на {{incode   -45 мм}}{: mediawiki}.
:   3.6. Нажмите кнопку **OK** для закрытия диалогового окна.

4\. Повторите этот процесс ещё раз, вставив второй, меньший куб, нажав на кнопку **Image:Part_Box.svg   16px Part_Box/ru**{: mediawiki}.

:   4.1. Второй куб будет создан с тем же именем, но с дополнительным номером, чтобы различать объекты.
:   4.2. Выберите {{incode   Куб001}}{: mediawiki} в древе проекта, и измените его размеры и положение как и с предыдущим объектом.
:   4.3. Измените **Length** на {{incode   80 мм}}{: mediawiki}.
:   4.4. Измените **Width** на {{incode   80 мм}}{: mediawiki}.
:   4.5. Измените **Height** на {{incode   80 мм}}{: mediawiki}.
:   4.6. Откройте диалог Placement.
:   4.7. Измените **X** на {{incode   -40 мм}}{: mediawiki}.
:   4.8. Измените **Y** на {{incode   -40 мм}}{: mediawiki}.
:   4.9. Измените **Z** на {{incode   -40 мм}}{: mediawiki}.
:   4.10. Нажмите кнопку **OK** для закрытия диалога.

## Объединение и вырезка 

10\. В древе проекта выберите {{incode   Куб001}}{: mediawiki}  и три цилиндра, затем нажмите **File:Part_Fuse.svg   16px Part_Fuse/ru**{: mediawiki}. Это создаст объект {{incode   Fusion}}{: mediawiki}.

11\. Затем выполните Булеву  обрезку {{incode   Куба}}{: mediawiki} и нового объекта {{incode   Fusion}}{: mediawiki}.

:   11.1. В древе проекта сначала выберите {{incode   Куб}}{: mediawiki}, а затем {{incode   Fusion}}{: mediawiki}.
:   11.2. Затем нажмите кнопку **File:Part_Cut.svg   16px Part_Cut/ru**{: mediawiki}. Это создаст объект {{incode   Cut}}{: mediawiki}.
:   
    **Примечание:**порядок, в котором вы выбираете объекты, важен для операции вырезания. Сначала выбирается базовый объект, а потом вырезаемый.
:   11.3. Если цвета выглядят странно, выберите новый объект вырезки {{incode   Cut}}{: mediawiki}, перейдите на вкладку **Вид**, нажмите на значение **Shape Color**, чтобы открыть диалоговое окно **Выбор цвета**, затем выберите серый цвет; также измените значение **Line Width** на {{incode   2.0}}{: mediawiki}.

! 
*Пустотелая форма, полученная вырезанием куба и трёх цилиндров из большего куба.*
