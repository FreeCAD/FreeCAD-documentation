## Введение

Этот учебник был первоначально написан Роландом Франком (†2017, r-frank), а затем переписан и проиллюстрирован vox.

Этот учебник создан чтобы научить вас использовать [Верстак Part](Part_Workbench/ru.md).

Верстак Part(Деталь) был первым разработанным верстаком. Он предоставляет основные геометрические элементы, которые могут быть использованы в качестве строительных блоков для других верстаков. Верстак Part предназначен для использования в традиционном рабочем процессе создания [конструктивной твердотельной геометрии](constructive_solid_geometry/ru.md)(CSG). Для более современного рабочего процесса с использованием эскизов, функции выдавливания и др. функций используйте [верстак PartDesign](PartDesign_Workbench/ru.md).

Вы попрактикуетесь в:

-   вставке примитивов
-   изменении параметров этих примитивных объектов
-   изменении их [размещения](placement/ru.md) (положения в пространстве)
-   выполнении Булевых(логических) операций

## Настройка

1\. Откройте FreeCAD и создайте новый пустой документ с помощью **Файл→ [Создать новый](File:Std_New.svg   16px]] [[Std_New/ru.md)**, и переключитесь на [верстак Part](Part_Workbench/ru.md).

:   1.1. Нажмите на кнопку**[изометрический Вид](File:Std_ViewIsometric.svg   16px]] [[Std_ViewIsometric/ru.md)**, или нажмите **0** на цифровой панели клавиатуры, для переключения в изометрический вид и лучшего представления 3D объектов.
:   1.2. Нажмите кнопку **[Уместить всё содержимое](File:Std_ViewFitAll.svg   16px]] [[Std_ViewFitAll/ru.md)** всякий раз, когда вы добавляете объекты для панорамирования и масштабирования [3D-вида](3D_view/ru.md), чтобы все элементы были видны на экране.
:   1.3. Удерживайте нажатой клавишу **Ctrl**, для выбора нескольких элементов. Если вы выбрали что-то неправильно или хотите отменить выбор всего, просто нажмите на пустое место в [3D-виде](3D_view/ru.md).



## Вставка примитивных кубов 

2\. Вставьте примитивный куб, нажав на кнопку **![](images/)**.

:   2.1. Выберите {{incode   Куб}} в [древе проекта](tree_view/ru.md).
:   2.2. Измените размеры во вкладке **Данные** на панели [Окна свойств](property_editor/ru.md).
:   2.3. Измените **Length(Длина)** на{{incode   90 мм}}.
:   2.4. Измените **Width(Ширина)** на {{incode   90 мм}}.
:   2.5. Измените **Height(Высота)** на {{incode   90 мм}}.

3\. На вкладке **Данные** панели [Окна свойств](property_editor/ru.md), нажмите на **Placement(Размещение)** так что бы появилась кнопка **...** справа.

:   3.1. Нажмите на неё для запуска диалогового окна [Placement(Размещение)](Std_Placement/ru.md).
:   3.2. Измените значения **Перемещения**.
:   3.3. Измените **X** на {{incode   -45 мм}}.
:   3.4. Измените **Y** на {{incode   -45 мм}}.
:   3.5. Измените **Z** на {{incode   -45 мм}}.
:   3.6. Нажмите кнопку **OK** для закрытия диалогового окна.

4\. Повторите этот процесс ещё раз, вставив второй, меньший куб, нажав на кнопку **![](images/)**.

:   4.1. Второй куб будет создан с тем же именем, но с дополнительным номером, чтобы различать объекты.
:   4.2. Выберите {{incode   Куб001}} в [древе проекта](tree_view/ru.md), и измените его размеры и положение как и с предыдущим объектом.
:   4.3. Измените **Length** на {{incode   80 мм}}.
:   4.4. Измените **Width** на {{incode   80 мм}}.
:   4.5. Измените **Height** на {{incode   80 мм}}.
:   4.6. Откройте диалог [Placement](Std_Placement.md).
:   4.7. Измените **X** на {{incode   -40 мм}}.
:   4.8. Измените **Y** на {{incode   -40 мм}}.
:   4.9. Измените **Z** на {{incode   -40 мм}}.
:   4.10. Нажмите кнопку **OK** для закрытия диалога.

:   5.1. Давайте выберем {{incode   Куб001}}, меньший куб в [древе проекта](tree_view/ru.md), и изменим его цвет. :   5.2. Теперь выберите {{incode   Куб}} больший куб, в [древе проекта](tree_view/ru.md). Для этого на вкладке **Вид**, измените значение **Transparency(Прозрачность)** на {{incode   70}}.

## Изменение визуальных свойств 

9\. Предыдущие операции создают три цилиндра, которые пересекаются друг с другом, а также пересекают кубы. Чтобы лучше увидеть это, мы можем изменить свойства представления в панели [окна свойств](property_editor/ru.md).

:   9.1. Выберите {{incode   Куб001}}, меньший куб, в древе проекта и измените прозрачность. На вкладке **Вид** измените значение **Transparency(Прозрачности)** на {{incode   70}}.

:   9.3. Выберите {{incode   Цилиндр001}}, на вкладке **Вид** нажмите на значение **Shape Color(Цвет формы)**, чтобы открыть диалоговое окно **Выбор цвета**, а затем выберите синий цвет.
:   9.4. Выберите {{incode   Цилиндр002}}, на вкладке **Вид** нажмите на значение **Shape Color(Цвет формы)**, чтобы открыть диалоговое окно **Выбор цвета**, а затем выберите розовый цвет.
:   9.5. Выберите все три цилиндра и на вкладке **Вид**, измените значение **Line Width(Ширины Линии)** на {{incode   2.0}}.

## Объединение и вырезка 

10\. В [древе проекта](tree_view/ru.md) выберите {{incode   Куб001}} (внутренний, меньший куб) и три цилиндра, затем нажмите **[Объединение(Fuse)](File:Part_Fuse.svg   16px]] [[Part_Fuse/ru.md)**. Это создаст объект {{incode   Fusion}}.

11\. Затем выполните Булеву (логическую) обрезку {{incode   Куба}}(большего куба) и нового объекта {{incode   Fusion(слияния)}}.

:   11.1. В [древе проекта](tree_view.md) сначала выберите {{incode   Куб}}, а затем {{incode   Fusion}}.
:   11.2. Затем нажмите кнопку **[Обрезать](File:Part_Cut.svg   16px]] [[Part_Cut/ru.md)**. Это создаст объект {{incode   Cut}}.
:   
    **Примечание:**порядок, в котором вы выбираете объекты, важен для операции вырезания. Сначала выбирается базовый объект, а потом вырезаемый.
:   11.3. Если цвета выглядят странно, выберите новый объект вырезки {{incode   Cut}}, перейдите на вкладку **Вид**, нажмите на значение **Shape Color(Цвета формы)**, чтобы открыть диалоговое окно **Выбор цвета**, затем выберите серый цвет; также измените значение **Line Width(ширины линии)** на {{incode   2.0}}.

![](images/03_T03_Part_cube_cut.png ) 
*Пустотелая форма, полученная вырезанием куба и трёх цилиндров из большего куба.*
