# Assembly3 ImportFromSTEP/ru

## Описание

Команда <img alt="" src=images/Assembly_Import.svg‎‎  style="width:24px;"> [Import from STEP](Assembly3_ImportFromSTEP/ru.md) открывает окно просмотра для выбора и выделения файлов STEP.

(Сборки STEP импортируются в единый документ.)

## Применение

1.  Press **<img src="images/Assembly_Import.svg‎‎" width=16px> [Import from STEP](Assembly3_ImportFromSTEP.md)** to open a file browser window
2.  Select the STEP file to import
