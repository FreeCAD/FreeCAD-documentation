# Sketcher CreateHeptagon/ru


</div>



## Описание


<div class="mw-translate-fuzzy">

Создает семиугольник, вписанный в окружность вспомогательной геометрии. При запуске инструмента указатель мыши меняется на белый крест с красным значком шестиугольника. Координаты указателя отображаются рядом с ним синим цветом в режиме реального времени.


</div>

![](images/SketcherCreateHeptagonExample.png‎ )



## Применение

See also: [Drawing aids](Sketcher_Workbench#Drawing_aids.md).


<div class="mw-translate-fuzzy">

-   После нажатия кнопки **[<img src=images/Sketcher_CreateHeptagon.svg style="width:16px"> [Семиугольник](Sketcher_CreateHeptagon.md)**, нажмите первый раз, чтобы задать центр, затем переместите мышь и нажмите второй раз, чтобы задать одну из вершин.
-   Нажатие **Esc** или правой кнопки мыши закрывает функцию.
