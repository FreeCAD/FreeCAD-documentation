El menú Ayuda proporciona acceso al sistema de ayuda y enlaces al sitio web y al foro de FreeCAD.



## Herramientas

Las siguientes herramientas están disponibles en este menú:

-   <img alt="" src=images/Std_OnlineHelp.svg  style="width:32px;"> [Ayuda](Std_OnlineHelp/es.md): Abre la documentación de ayuda.

-   <img alt="" src=images/Std_FreeCADWebsite.svg  style="width:32px;"> [Sitio web de FreeCAD](Std_FreeCADWebsite/es.md): Abre el sitio web principal del proyecto FreeCAD.

-   <img alt="" src=images/Std_FreeCADDonation.svg  style="width:32px;"> [Donar](Std_FreeCADDonation/es.md): Abre la página de la wiki de FreeCAD Donar.

-   <img alt="" src=images/Std_FreeCADUserHub.svg  style="width:32px;"> [Documentación de usuarios](Std_FreeCADUserHub/es.md): Abre la página wiki de documentación de usuarios de FreeCAD.

-   <img alt="" src=images/Std_FreeCADPowerUserHub.svg  style="width:32px;"> [Documentación de Python scripting ](Std_FreeCADPowerUserHub/es.md): Abre la página wiki de documentación para usuarios avanzados de FreeCAD.

-   <img alt="" src=images/Std_PythonHelp.svg  style="width:32px;"> [Documentación automática de módulos de Python.](Std_PythonHelp/es.md): Abre el índice de módulos de Python en su navegador de Internet predeterminado.

-   <img alt="" src=images/Std_FreeCADForum.svg  style="width:32px;"> [Foro de FreeCAD](Std_FreeCADForum/es.md): Abre el foro de FreeCAD.

-   <img alt="" src=images/Std_FreeCADFAQ.svg  style="width:32px;"> [FreeCAD FAQ](Std_FreeCADFAQ/es.md): Abre la página wiki de preguntas frecuentes de FreeCAD.

-   <img alt="" src=images/Std_ReportBug.svg  style="width:32px;"> [Informar de un error](Std_ReportBug.md): Abre la página de problemas de FreeCAD GitHub.

-   <img alt="" src=images/Std_About.svg  style="width:32px;"> [Acerca de FreeCAD](Std_About/es.md): Abre un cuadro de diálogo con información sobre FreeCAD.

-   <img alt="" src=images/Std_WhatsThis.svg  style="width:32px;"> [¿Qué es esto?](Std_WhatsThis/es.md): Abre la documentación de ayuda para un comando.

-   <img alt="" src=images/Std_RestartInSafeMode.svg  style="width:32px;"> [Restart in safe mode](Std_RestartInSafeMode.md): Restarts FreeCAD in a *factory reset* state for debugging. <small>(v1.0)</small> 

-   <img alt="" src=images/Start_Start.svg  style="width:32px;"> [Start](Start_Start.md): Opens the Start page. <small>(v1.0)</small>
