## Description

The <img alt="" src=images/Sketcher_CreateOctagon.svg  style="width:24px;"> [Sketcher CreateOctagon](Sketcher_CreateOctagon.md) tool creates an octagon. <small>(v1.0)</small> : This is the same tool as [Sketcher CreateRegularPolygon](Sketcher_CreateRegularPolygon.md) but with the number of sides preset to 8.

![](images/SketcherCreateOctagonExample.png‎ )

## Usage

See also: [Drawing aids](Sketcher_Workbench#Drawing_aids.md).

1.  There are several ways to invoke the tool:
    -   Press the **<img src="images/Sketcher_CreateOctagon.svg" width=16px> [Octagon](Sketcher_CreateOctagon.md)** button.
    -   Select the **Sketcher → Sketcher geometries → <img src="images/Sketcher_CreateOctagon.svg" width=16px> Create octagon** option from the menu.
    -   Use the keyboard shortcut: **G** then **P**, then **8**.
2.  For further steps see [Sketcher CreateRegularPolygon](Sketcher_CreateRegularPolygon#Usage.md).
