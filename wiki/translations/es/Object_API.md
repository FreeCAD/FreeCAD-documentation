# Object API/es

 {{VeryImportantMessage|(October 2019) Do not edit this page. The information is incomplete and outdated. For the latest API, see the [https://www.freecadweb.org/api autogenerated API documentation], or generate the documentation yourself, see [Source documentation](Source_documentation.md).}}

Siendo paramétricos, los objetos del documento en FreeCAD pueden tener diversas propiedades adicionales, pero éstas son las básicas, presentes en cada objeto del documento de FreeCAD. Los objetos se pueden recuperar fácilmente por su nombre. Por ejemplo: 
```python
myObj = FreeCAD.ActiveDocument.myObjectName
print myObj.PropertiesList
```


{{APIProperty|Content|Una representación XML de las propiedades de un objeto.}}


{{APIProperty|Label|Obtiene/establece la etiqueta de los objetos. La cadena de texto puede ser unicode.}}


{{APIProperty|Name|El nombre único de un objeto.}}


{{APIProperty|Placement|Obtiene/establece la ubicación de un objeto. Una ubicación define una orientación (rotación) y una posición (base) en el espacio 3D. Se utiliza cuando es necesario no escalar u otra distorsión.}}


{{APIProperty|PropertiesList|Una lista de las propiedades de un objeto.}}


{{APIProperty|State| El estado de FreeCAD de un objeto (por ejemplo si necesita ser recalculado).}}


{{APIProperty|Type|Una cadena de texto describiendo el tipo del objeto.}}


{{APIProperty|ViewObject|El Proveedor de Vista asociado (objeto FreeCADGUI) a un objeto.}}


{{APIFunction|getAllDerivedFrom| | |Todos los descendientes de este objeto.}}


{{APIFunction|getDocumentationOfProperty| | |La cadena de texto de documentación de la propiedad de esta clase.}}


{{APIFunction|getGroupOfProperty| | |El nombre del grupo al cual pertenece el objeto en esta clase. Las propiedades ordenadas en diferentes grupos denominados según convenga.}}


{{APIFunction|getPropertyByName| | |El valor de una propiedad denominada.}}


{{APIFunction|getTypeOfProperty| | |tEl tipo de una propiedad denominada. Esto puede ser (Hidden,ReadOnly,Output) o cualquier combinación.}}


{{APIFunction|isDerivedFrom| | |True si el tipo dado es un padre.}}


{{APIFunction|purgeTouched| |Marca el objeto como unchanged| }}


{{APIFunction|touch| |Marca el objeto como changed (tocado)| }}
