# Sketcher CreateHeptagon

## Description

The <img alt="" src=images/Sketcher_CreateHeptagon.svg  style="width:24px;"> [Sketcher CreateHeptagon](Sketcher_CreateHeptagon.md) tool creates a heptagon. <small>(v1.0)</small> : This is the same tool as [Sketcher CreateRegularPolygon](Sketcher_CreateRegularPolygon.md) but with the number of sides preset to 7.

 ![](images/SketcherCreateHeptagonExample.png‎ ) 

## Usage

See also: [Drawing aids](Sketcher_Workbench#Drawing_aids.md).

1.  There are several ways to invoke the tool:
    -   Press the **<img src="images/Sketcher_CreateHeptagon.svg" width=16px> [Heptagon](Sketcher_CreateHeptagon.md)** button.
    -   Select the **Sketcher → Sketcher geometries → <img src="images/Sketcher_CreateHeptagon.svg" width=16px> Create heptagon** option from the menu.
    -   Use the keyboard shortcut: **G** then **P**, then **7**.
2.  For further steps see [Sketcher CreateRegularPolygon](Sketcher_CreateRegularPolygon#Usage.md).
