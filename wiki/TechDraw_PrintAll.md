# TechDraw PrintAll

## Description

The <img alt="" src=images/TechDraw_PrintAll.svg  style="width:24px;"> **TechDraw PrintAll** tool sends all pages in a document to the printer.

## Usage

1.  There are several ways to invoke the tool:
    -   Press the **<img src="images/TechDraw_PrintAll.svg" width=16px> [Print All Pages](TechDraw_PrintAll.md)** button.
    -   Select the **TechDraw → Page → <img src="images/TechDraw_PrintAll.svg" width=16px> Print All Pages** option from the menu.
    -   If a page is displayed in the [Main view area](Main_view_area.md): right-click the page\'s window and select the **Print All Pages** option from the context menu.

## Notes

-   Use [Std Print](Std_Print.md) to print a single page.
