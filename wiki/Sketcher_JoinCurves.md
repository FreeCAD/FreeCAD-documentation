# Sketcher JoinCurves

## Description

The <img alt="" src=images/Sketcher_JoinCurves.svg  style="width:24px;"> [Sketcher JoinCurves](Sketcher_JoinCurves.md) tool creates a B-spline by joining two existing [B-splines](B-Splines.md) or other edges.

## Usage

1.  Do one of the following:
    -   Select an end point of two B-splines or other edges.
    -   Select a [coincident point](Sketcher_ConstrainCoincident.md) connecting two B-splines or other edges.
2.  There are several ways to invoke the tool:
    -   Press the **<img src="images/Sketcher_JoinCurves.svg" width=16px> [Join curves](Sketcher_JoinCurves.md)** button.

    -   Select the **Sketch → Sketcher B-spline tools → <img src="images/Sketcher_JoinCurves.svg" width=16px> Join curves** option from the menu.

    -   
        <small>(v1.0)</small> 
        
        : Right-click in the [3D view](3D_view.md) and select the **<img src="images/Sketcher_JoinCurves.svg" width=16px> Join curves** option from the context menu.
