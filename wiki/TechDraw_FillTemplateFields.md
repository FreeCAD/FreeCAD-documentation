# TechDraw FillTemplateFields

## Description

The <img alt="" src=images/TechDraw_FillTemplateFields.svg  style="width:24px;">[FillTemplateFields](TechDraw_FillTemplateFields.md) tool fills the editable text fields in the title block of the current page.

## Usage

1.  There are several ways to invoke the tool:
    -   Press the **<img src="images/TechDraw_FillTemplateFields.svg" width=16px> [Update template fields](TechDraw_FillTemplateFields.md)** button.
    -   Select the **TechDraw → Page → <img src="images/TechDraw_FillTemplateFields.svg" width=16px> Update template fields** option from the menu.
2.  \...

## Notes

This tool seems to have nothing to do with the newly implemented **freecad:autofill** attribute in FreeCAD\'s SVG namespace that appeared around the same time.
