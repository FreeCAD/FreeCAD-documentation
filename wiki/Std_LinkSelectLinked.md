# Std LinkSelectLinked

## Description

The **Std LinkSelectLinked** command selects the **Linked Object**, the source object, of an [App Link](App_Link.md) object, a link.

## Usage

1.  Select a link.
2.  Select the **Link actions â†’ <img src="images/Std_LinkSelectLinked.svg" width=16px> Go to linked object** option from the [Tree view](Tree_view.md) context menu.
3.  The linked object is selected. If this object belongs to an external document that document is activated.
4.  Optionally use **<img src="images/Std_SelBack.svg" width=16px> [Std SelBack](Std_SelBack.md)** to reselect the link.
