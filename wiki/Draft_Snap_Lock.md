# Draft Snap Lock

## Description

The <img alt="" src=images/Draft_Snap_Lock.svg  style="width:24px;"> **Draft Snap Lock** command enables or disables snapping globally.

## Usage

For general information about snapping see [Draft Snap](Draft_Snap.md).

1.  To invoke **Draft Snap Lock** do one of the following:
    -   Press the **<img src="images/Draft_Snap_Lock.svg" width=16px> [Snap lock](Draft_Snap_Lock.md)** button in the Draft snap toolbar.
    -   [Draft](Draft_Workbench.md): Press the **<img src="images/Draft_Snap_Lock.svg" width=x16px><img src="images/Toolbar_flyout_arrow.svg" width=x16px>** button (short click) in the [Draft snap widget](Draft_snap_widget.md). <small>(v0.21)</small> 
    -   Draft: Hold down the **<img src="images/Draft_Snap_Lock.svg" width=x16px><img src="images/Toolbar_flyout_arrow.svg" width=x16px>** button in the Draft snap widget and in the menu that opens select the **<img src="images/Draft_Snap_Lock.svg" width=16px> Snap lock** option.
    -   [BIM](BIM_Workbench.md): Select the **Snapping → <img src="images/Draft_Snap_Lock.svg" width=16px> Snap lock** option from the menu, or from the [3D view](3D_view.md) context menu.
    -   Use the keyboard shortcut: **Shift**+**S**. If a command is active this shortcut only works if none of the input boxes in the task panel has the focus. <small>(v0.21)</small> 
    -   Use the \"in-command\" shortcut: **S**. This shortcut can only be used if a command is active and one of the input boxes in the task panel has the focus. It does not work for all commands.
