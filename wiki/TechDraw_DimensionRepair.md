# TechDraw DimensionRepair

## Description

The Dimension Repair tool is used to adjust the reference points of dimensions. These references may be rendered incorrect due to the \"[topological naming](topological_naming_problem.md)\" or Hidden Line Removal. The tool can correct references based on 2d or 3d geometry.

See [TechDraw LengthDimension](TechDraw_LengthDimension#Limitation.md) for more on dimensions and topological naming.

## Usage

1.  Select the dimension to be corrected and, optionally, the new reference Vertices or Edges on the View or in the [3D view](3D_view.md).
2.  Press the **<img src="images/TechDraw_DimensionRepair.svg" width=16px> [Dimension Repair](TechDraw_DimensionRepair.md)
** button or **TechDraw → Repair Dimension References**
3.  A dialog will be presented showing the dimension being repaired and the current reference geometry (or the preselected geometry).
4.  Select the replacement geometry and press the \"Use current selection\" button.
5.  The dimension will be updated on pressing OK.

## Properties

Dimension Repair does not introduce any new properties.

## Scripting

See also: [Autogenerated API documentation](https://freecad.github.io/SourceDoc/) and [FreeCAD Scripting Basics](FreeCAD_Scripting_Basics.md).

The Dimension Repair tool is not used in [macros](Macros.md) or from the [Python](Python.md) console. Dimension reference properties can be updated using Python.
