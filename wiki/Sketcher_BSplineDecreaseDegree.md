# Sketcher BSplineDecreaseDegree

## Description

The <img alt="" src=images/Sketcher_BSplineDecreaseDegree.svg  style="width:24px;"> [Sketcher BSplineDecreaseDegree](Sketcher_BSplineDecreaseDegree.md) tool decreases the degree (order) of [B-splines](B-Splines.md).

## Usage

1.  Select one or more B-splines.
2.  There are several ways to invoke the tool:
    -   Press the **<img src="images/Sketcher_BSplineDecreaseDegree.svg" width=16px> [Decrease B-spline degree](Sketcher_BSplineDecreaseDegree.md)** button.

    -   Select the **Sketch → Sketcher B-spline tools → <img src="images/Sketcher_BSplineDecreaseDegree.svg" width=16px> Decrease B-spline degree** option from the menu.

    -   
        <small>(v1.0)</small> 
        
        : Right-click in the [3D view](3D_view.md) and select the **<img src="images/Sketcher_BSplineDecreaseDegree.svg" width=16px> Decrease B-spline degree** option from the context menu.
