# Part DatumLine

## Description

The **Part DatumLine** command creates a datum line object that can be attached to other objects. A datum line is one of several [datum objects](Std_Base#Part_Datums.md). A datum object is typically used to attach multiple other objects to. If the position or orientation of a datum object changes, all objects attached to it will follow.
