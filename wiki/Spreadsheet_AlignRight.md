# Spreadsheet AlignRight

## Description

The <img alt="" src=images/Spreadsheet_AlignRight.svg  style="width:24px;"> **Spreadsheet AlignRight** tool aligns the contents of selected cells to the right.

## Usage

1.  Make sure there is an active [spreadsheet](Spreadsheet_CreateSheet.md).
2.  Select one or more cells.
3.  There are several ways to invoke the tool:
    -   Press the **[<img src=images/Spreadsheet_AlignRight.svg style="width:16px"> [Align right](Spreadsheet_AlignRight.md)** button.
    -   Select the **Spreadsheet → Alignment → [<img src=images/Spreadsheet_AlignRight.svg style="width:16px"> Align right** option from the menu.
